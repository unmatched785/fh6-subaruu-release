# 2026-06-29 - 단일 실행파일/초보자 가이드/테스트런 기준 정리 빌드

이번 버전은 배포 폴더를 더 찾기 쉽게 줄이고, 초보자가 틀린 화면에서 `F1`을 누르는 사고를 줄이기 위한 안내를 강화한 빌드입니다.

## 다운로드

- 파일: `subaruu.zip`
- SHA256: `75C74F1C30A45632162B741580EDFB77765DA89C3B354E1007168A64334FC1E3`

## 이번 버전에서 바뀐 점

- .NET 런타임 DLL들을 `subaruu.exe` 단일 실행파일 안으로 묶었습니다.
- 배포 ZIP의 최상위 항목을 크게 줄여서 사용자가 `subaruu.exe`를 찾기 쉽게 했습니다.
- `eventlab-templates` 폴더는 이미지 인식에 필요하므로 `subaruu.exe` 옆에 유지합니다.
- 초보자용 `GUIDE.md`를 추가했습니다.
- README를 빠른 시작, 긴급 정지, 정상 시작 위치 중심으로 정리했습니다.
- 공개 배포용 기본 아이콘을 오리지널 파란 랠리카 아이콘으로 교체했습니다.
- 테스트런 기준을 정리했습니다.
  - `최소런`: 런 1회, 구매 2대, 강화 2대, 삭제 1대, 회차 1회
  - `권장런`: 런 2회, 구매 3대, 강화 3대, 삭제 2대, 회차 2회
- 구매 시작 위치는 `자동차 컬렉션 > 제조사 > 자동차 선택 화면`으로 명시했습니다.
- 자동차 매장/상품권/스토어/현금 결제 화면이 나오면 잘못된 경로이므로 즉시 `F2`로 중지하라고 안내했습니다.
- 필수 게임 설정을 문서에 추가했습니다.
  - `주행 보조 사전 설정`: `보조 최대`
  - `HUD 및 게임플레이 > 기술`: `꺼짐`
  - `해상도`: `FHD, 1920 x 1080`
- 슈퍼휠 탭의 `스바루작`, `라부엘토작`, `마즈다작`이 실행 버튼이 아니라 프리셋 버튼이라는 설명을 추가했습니다.
- 한국 시간 기준 매주 목요일 23:30 전후에는 새 시즌/차량 갱신으로 제조사 위치와 차량 위치가 바뀔 수 있으므로, 기존 이동횟수를 믿지 말고 작은 테스트런부터 다시 확인하라는 경고를 추가했습니다.
- 스바루작 이벤트랩 맵 코드 `113938786`, 작업차 튜닝 코드 `146241846`, 스바루작 맵만 따로 즐겨찾기하는 권장 설정을 문서에 추가했습니다.
- 릴리즈 다운로드 에셋 이름을 `subaruu.zip`으로 단순화하고, `.sha256` 파일은 에셋에서 제거했습니다. SHA256 값은 릴리즈 본문에만 남깁니다.
- 스바루 작업차 준비용 핵심 자동차 마스터리 6개 설명을 추가했습니다.
  - `멀티 맥서`, `즐거움의 시작`, `실력을 보여줘`, `밥상 엎기`, `격파 뺑소니`, `목숨 추가`
- 스바루런 주행 가속 입력을 `W` 단독에서 `W + ↑ 방향키` 동시 입력으로 바꿨습니다.
- 배포 ZIP 안쪽 폴더명도 긴 빌드명 대신 `subaruu`로 단순화했습니다.

## 테스트런 기준 설명

`최소런`은 4분짜리 100포인트 맵을 1회 돌았을 때의 기본 확인용입니다.
레부엘토작은 1대당 39포인트라서 100포인트로 최대 2대까지 강화할 수 있고, 삭제 1대까지 테스트할 수 있습니다.

`권장런`은 2회차 요소를 넣어서 다음 회차 재진입과 이어서 작동하는 흐름까지 확인합니다.

## 유지된 점

- 맵 코드가 비어 있으면 `내 즐겨찾기` 이벤트랩 경로를 사용합니다.
- 작업차 선택은 이미지 인식과 키보드 이동을 우선 사용합니다.
- 배포 ZIP에는 `subaruu.conf`를 포함하지 않습니다. 기존 사용자 설정을 덮어쓰지 않기 위한 처리입니다.

## 검증한 내용

- Windows x64 self-contained single-file 배포 ZIP 생성 확인
- ZIP 안에 `subaruu.conf`가 포함되지 않는 것 확인
- ZIP 안에 `subaruu.exe`, `README.md`, `GUIDE.md`, `subaruu.default.conf`, `eventlab-templates`가 포함된 것 확인
- 패키지 SHA256 계산 완료
