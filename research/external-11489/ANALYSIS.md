# External 11,489-slot wheelspin sample analysis

작성일: 2026-07-04

이 문서는 외부 11,489-slot 휠스핀 표본을 해석하기 위한 요약 문서입니다.
기존 개인 실측 문서인 `WHEELSPIN_RESEARCH.md`와는 별개 자료입니다.

## 표본 성격

원본 표본은 아래 조건에서 기록되었습니다.

```text
총 표본 = 11,489 slots
Super Wheelspin은 3 slots로 계산
Clothes / Horns / Emotes를 모두 얻은 뒤 기록
획득 가능한 차량을 최소 1대씩 보유한 뒤 기록
```

따라서 이 표본은 보상 슬롯 단위의 대형 혼합 표본입니다.
일반휠과 슈퍼휠이 분리되어 있지 않으므로, 일반 슬롯과 슈퍼 슬롯의 상대 비율 `W/S`를 직접 계산하는 데 쓰면 안 됩니다.

핵심 기준값:

```text
Legendary Car = 182 / 11,489 = 1.584%
FE            = 182 / 11,489 = 1.584%
Legendary+FE  = 364 / 11,489 = 3.168%
```

개인 실측의 LC 관측률은 이 외부 표본과 크게 어긋나지 않습니다.

```text
외부 표본 LC = 1.584%
개인 실측 일반 LC = 13 / 808 = 1.609%
개인 실측 슈퍼 LC = 18 / 1068 = 1.685%
```

이 비교는 개인 실측의 확률 크기가 비정상적으로 높거나 낮지 않다는 참고 근거입니다.
다만 외부 표본은 일반/슈퍼 분리 표본이 아니므로, 개인 실측의 `W/S` 계산에 합산하면 안 됩니다.

## WX 정의

이 문서에서는 휠스핀 보상 차량을 세 단계로 나눕니다.

```text
WX_source = Wheelspin / Seasonal 보상 차량 45종 전체
WX_non_FE = WX_source 중 Forza Edition을 제외한 차량
FE = Forza Edition 차량
```

실전 수집 관점에서는 `WX_non_FE`와 `FE`를 분리해서 보는 편이 낫습니다.
FE는 휠스핀 보상 풀에 들어 있지만, 개인 연구에서는 목표 가치가 낮은 별도 범주로 취급했기 때문입니다.

## CR 해석

외부 표본의 평균값은 라부엘토작 CR 손익 해석에 중요한 기준을 줍니다.

라부엘토작 1사이클은 슬롯 기준으로 아래와 같습니다.

```text
일반 슬롯 75 + 슈퍼 슬롯 75 = 총 150 slots
```

외부 혼합 평균을 단순 대입하면 아래처럼 됩니다.

| 기준 | 150 slots 기대값 | 라부엘토 구매비 대비 |
| --- | ---: | ---: |
| 모든 차량 즉시판매 | 14.16M CR | 흑자 |
| Legendary/FE 보유 | 7.71M CR | 적자 가능 |
| CR only | 5.97M CR | 적자 |

라부엘토 구매비는 8,668,750 CR입니다.
따라서 라부엘토작의 지갑 CR 손익은 Legendary/FE/WX 차량을 팔 것인지 보유할 것인지에 따라 크게 달라집니다.

이것은 개인 실측에서 관측된 현상과 일치합니다.

```text
수집 가치가 있는 차를 보유하면 지갑 CR은 줄 수 있음
모든 중복차를 즉시판매하면 장기 평균은 흑자에 가까워짐
```

## 연구에서의 사용법

이 외부 표본은 아래 용도로 사용합니다.

```text
1. 전체 슬롯 기준 LC 확률 기준값
2. 전체 슬롯 기준 FE 확률 기준값
3. 전체 슬롯 기준 WX_non_FE 확률 기준값
4. CR 기대값에서 차량 가치가 차지하는 비중 확인
5. 개인 실측값이 비정상적이지 않은지 확인하는 참고 기준
```

사용하면 안 되는 용도:

```text
1. 일반휠과 슈퍼휠의 직접 비교
2. 라부엘토작과 마즈다작의 W/S 직접 계산
3. 개인 실측 표본과 단순 합산
```

## 결론

외부 11,489-slot 표본은 개인 실측의 결론을 직접 대체하지 않습니다.
하지만 전체 휠스핀 보상 분포의 큰 기준값으로는 가치가 큽니다.

핵심은 아래입니다.

```text
LC = 약 1.584% / slot
FE = 약 1.584% / slot
WX_non_FE = 약 1.340% / slot
WX_source 전체 = 약 2.925% / slot
```

개인 실측의 LC 관측률은 외부 대형 표본의 LC 확률과 가까운 범위에 있습니다.
따라서 현재 개인 실측이 표본은 아직 작더라도, 확률 규모 자체는 외부 대형 표본과 크게 충돌하지 않습니다.
