# v0.6.0-rc.1

## 핵심 시험 기능

- 브라우저가 메인 모니터에서 FH6를 완전히 가린 상태를 목표로 하는 `백그라운드 실험` 체크박스를 추가했습니다.
- F1 뒤 3초 준비 시간을 제공하고 코드 숫자, 검색 Enter, 메뉴 이동, W, Esc, Enter를 FH6 창에 직접 보냅니다.
- 사용자는 실행 중 브라우저의 키보드와 마우스를 평범하게 사용할 수 있습니다.

## 안전 제한

- FH6 `포커스 상실 시 일시 정지`를 끄고 전체 창 모드 또는 창 모드를 사용하세요.
- FH6를 최소화하지 마세요. 브라우저로 완전히 가리는 것은 이번 시험 범위입니다.
- 백그라운드 모드에서는 결과 OCR이 강제로 꺼집니다.
- 스바루런과 테스트-챌린지만 허용됩니다.
- 테스트 회차는 1, 구매·강화·삭제는 0으로 고정됩니다.
- 구매·마스터리·삭제·휠스핀·올인원 탭은 잠깁니다.
- 화면 캡처, 모든 OCR, 실제 마우스 클릭, FH6 전경 전환이 호출되면 안전 정지합니다.
- 창 메시지 전송 실패나 FH6 최소화를 감지하면 안전 정지합니다.
- 완료·중지·오류 때 W와 주요 키를 해제하고 체크박스를 자동으로 끕니다.

## 미검증 항목

- FH6가 포커스를 잃은 상태에서 직접 게시된 W를 주행 입력으로 계속 처리하는지
- 백그라운드에서 챌린지 코드 숫자와 검색 Enter를 처리하는지
- Xbox Game Pass/게임패드 가상 키보드 환경에서도 같은 경로가 동작하는지

이 RC는 위 항목의 실게임 검증을 위한 버전입니다. 처음에는 테스트 탭에서 런 1, 구매·강화·삭제 0으로 확인하세요.
