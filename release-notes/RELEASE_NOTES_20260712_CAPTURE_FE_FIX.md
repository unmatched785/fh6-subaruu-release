# 휠스핀 저장공간·FE 오탐 보정 2026-07-12

정상 휠스핀 화면이 매회 PNG로 누적되는 문제와 일반 보라 차량을 FE로 오인할 수 있던 문제를 수정했습니다.

## 변경 사항

- `상세 화면 저장` 옵션 추가, 기본 OFF
- 옵션 OFF에서는 정상 결과·팝업·포커스·다음 회전 화면을 저장하지 않음
- 오류·안전정지·F4 수동 캡처는 옵션과 관계없이 항상 저장
- 옵션 ON에서는 기존처럼 정상 진행 화면도 저장
- FE 시각 판정은 연구 자료의 8종 연식·제조사 후보에만 적용
- 일반 보라 `1985 Toyota Sprinter Trueno` FE 오탐 수정
