이전 2026-07-04 경매 시작 포커스 보정 빌드까지 포함한 최신 권장 빌드입니다.

## 다운로드

- Asset: `subaruu.zip`
- SHA256: `3DE978CE13B3CF07A73A89829EF9831D6E29692619DF5AF105D6B12771FBC208`

## 변경

- `테스트`와 `올인원` 탭에 `보유CR` 입력칸을 추가했습니다.
- 보유 CR을 입력하지 않으면 테스트런과 올인원이 시작되지 않습니다.
- 입력값은 숫자만 사용하며 최대 `999999999`까지 허용합니다.
- 프리셋 차량 가격 기준으로 `차량 가격 x 구매 대수 x 회차`를 계산합니다.
- CR이 부족하면 가능한 회차 또는 구매/강화/삭제 대수로 자동 제한하고, 차량 1대도 살 수 없으면 실행을 막습니다.

## 기준 차량 가격

- 스바루작: `81,700 CR`
- 라부엘토작: `346,750 CR`
- 마즈다작: `95,000 CR`

## 확인

- `dotnet build -c Release`
- `dotnet publish -c Release -r win-x64 --self-contained true`
- 보유CR 붙여넣기 숫자 정규화 확인
- 10자리 이상 입력 시 `999999999` 제한 확인
- ZIP 안쪽 루트 폴더가 `subaruu/` 하나인 것 확인
- ZIP 안에 `subaruu.conf`, `subaruu.conf.bak`, `log` 폴더가 포함되지 않는 것 확인
- 패키지 SHA256 계산 완료

## 문제 보고

문제가 생기면 `F2`로 즉시 멈추고, `subaruu.exe` 옆의 `log` 폴더와 `subaruu.conf`를 같이 보내주세요.
`last-run-status.txt`도 `log` 폴더 안에 함께 저장됩니다.
