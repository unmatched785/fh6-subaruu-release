## 변경

- 맵 코드 입력을 클립보드 붙여넣기 대신 숫자 직접 입력 방식으로 바꿨습니다.
- 입력칸 초기화는 `End -> Shift+Home -> Backspace`로 처리해서 한글 IME 조합으로 `ㅁㅁㅁㅁ` 같은 문자가 들어가는 상황을 피합니다.
- 숫자 입력 간격은 안정성을 위해 50ms로 맞췄습니다.
- 이벤트랩 작업차 탐색에서 실제 기본 템플릿과 1.00 배율을 먼저 보도록 바꿔 인식 시작 시간을 줄였습니다.
- 작업차 탐색 로그에 페이지별 탐색 시간, 누적 시간, 템플릿 이름, 점수를 남깁니다.
- 자동차 마스터리 진입 기본 경로를 현재 메뉴 기준 `↓7`로 줄였습니다.
- 경매장 최근 추가됨 필터 이동 간격을 100ms로 줄였습니다.
