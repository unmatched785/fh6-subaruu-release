# 독립 휠스핀 자동 처리 통합 2026-07-12

검증된 2026-07-11 안정판에 일반·슈퍼 휠스핀 독립 실행 기능을 추가한 공개 릴리스입니다.

## 주요 변경

- 기존 `슈퍼휠` 탭 이름을 실제 역할에 맞게 `마스터리`로 변경
- 새 `휠스핀` 탭에서 일반 또는 슈퍼 휠스핀을 목표 횟수만큼 처리
- 빠른 정지 후 슬롯 색상과 `CR/차량/비차량`을 판정
- 독점 차량은 등급과 관계없이 보관
- `레전더리 보관` 기본 활성화, `에픽 보관` 선택 제공
- 보관 대상이 아닌 비독점 중복 차량만 CR로 판매
- CR·의상·경적은 판매 대상으로 취급하지 않고 그대로 수령
- 상태 줄에 슬롯 색상, 품목 유형, 독점 여부와 보관·판매 결정을 표시
- 일반·슈퍼 모두 목표 완료 후 휠 메뉴에 머무름

## 기존 자동화와의 관계

- 스바루런·구매·마스터리·삭제·테스트·올인원은 2026-07-11 안정판 흐름 유지
- 새 휠스핀 기능은 독립 실행 전용
- 테스트런·올인원에는 아직 휠스핀이 연결되어 있지 않음

## 설정 및 배포 정리

- 실행 파일명과 ZIP 이름은 기존과 동일하게 `subaruu.exe`, `subaruu.zip` 사용
- 새 설치 기본 주행 시간은 245초, 보정 주행은 120초 유지
- 기존 루트 `subaruu.conf`를 새 `config/subaruu.conf`로 자동 이관
- 배포 ZIP에는 사용자 설정과 로그를 넣지 않음
- OCR 실행에 필요한 `data/tessdata`와 `x64`를 함께 배포
- 폐기된 `GUIDE.md`는 배포 ZIP에서 제외하고 README를 현재 기준 문서로 사용

## 문서 기준 복원

- 작업차 튜닝 코드 `171 460 155`
- 화면 모드 `전체 화면` 또는 `창 전체 화면`
- FHD 정상 주행 약 4분 1초대

- Release 빌드 오류 0개
