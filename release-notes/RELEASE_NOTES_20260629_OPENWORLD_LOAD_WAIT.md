# 2026-06-29 - 오픈월드 복귀 대기 핫픽스

이번 버전은 스바루런 후 오픈월드로 나왔는데 구매 메뉴로 들어가지 못하고, 이후 단계가 꼬이는 문제를 줄이기 위한 핫픽스입니다.

## 다운로드

- 파일: `subaruu.zip`
- SHA256: `2B74038B01E539B593DD6E049C790D87DF3C2CBD6E3E45C88C44F1882AA274C5`

## 이번 버전에서 바뀐 점

- 보정런 후 오픈월드로 복귀하는 마지막 대기 시간이 기존 20초 고정값이었습니다.
- 이제 이 대기 시간도 `스바루런` 탭의 `로딩 대기` 값을 사용합니다.
- 기본 `로딩 대기` 값을 20초에서 30초로 올렸습니다.
- 느린 PC에서는 `로딩 대기`를 30~45초로 늘려서 테스트할 수 있습니다.
- GUIDE에 “구매가 안 됐는데 슈퍼휠/삭제 단계로 넘어가는 것처럼 보이면 즉시 F2” 경고를 추가했습니다.

## 유지된 점

- 다운로드 파일명은 계속 `subaruu.zip` 하나만 사용합니다.
- ZIP 안쪽 폴더명은 계속 `subaruu`입니다.
- Windows x64 self-contained single-file 빌드입니다.
- 배포 ZIP에는 `subaruu.conf`를 포함하지 않습니다.
- 스바루런 `W + ↑ 방향키` 동시 가속 입력 핫픽스는 유지됩니다.

## 검증한 내용

- Release 빌드 성공
- Windows x64 self-contained single-file 배포 ZIP 생성 확인
- ZIP 안쪽 루트 폴더가 `subaruu/` 하나인 것 확인
- ZIP 안에 `subaruu.conf`가 포함되지 않는 것 확인
- ZIP 안의 `subaruu.default.conf` 기본 `EventLabLoadWait=30` 확인
- 패키지 SHA256 계산 완료

## 주의

기존 사용자의 `subaruu.conf`는 배포 ZIP으로 덮어쓰지 않습니다.
이미 설정 파일이 있는 사용자는 필요하면 직접 `스바루런` 탭의 `로딩 대기`를 30~45초로 늘려야 합니다.

이 도구는 비공식 개인 편의 도구입니다.
사용은 전적으로 사용자 본인 책임입니다.
