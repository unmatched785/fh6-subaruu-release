# 2026-06-29 - 스바루런 W+↑ 가속 입력 핫픽스

이번 버전은 스바루런 주행 중 가속 입력을 더 넓게 먹도록 바꾼 핫픽스입니다.
기존 `W` 단독 입력에서 `W + ↑ 방향키` 동시 입력으로 변경했습니다.

## 다운로드

- 파일: `subaruu.zip`
- SHA256: `75C74F1C30A45632162B741580EDFB77765DA89C3B354E1007168A64334FC1E3`

## 이번 버전에서 바뀐 점

- 스바루런 주행 가속 입력을 `W` 단독에서 `W + ↑ 방향키` 동시 입력으로 바꿨습니다.
- 스바루런 보정 주행 구간도 같은 방식으로 `W + ↑ 방향키`를 함께 누릅니다.
- 긴급 정지 시 `W`뿐 아니라 `↑ 방향키`도 함께 떼도록 했습니다.
- README/GUIDE에 스바루런 주행 입력 방식을 명시했습니다.
- 배포 ZIP 안쪽 폴더명을 긴 빌드명 대신 `subaruu`로 단순화했습니다.

## 유지된 점

- 다운로드 파일명은 계속 `subaruu.zip` 하나만 사용합니다.
- Windows x64 self-contained single-file 빌드입니다.
- 배포 ZIP에는 `subaruu.conf`를 포함하지 않습니다.
- 기존 필수 설정, 테스트런 기준, 작업차 준비 가이드는 유지됩니다.

## 검증한 내용

- Windows x64 self-contained single-file 배포 ZIP 생성 확인
- ZIP 안쪽 루트 폴더가 `subaruu/` 하나인 것 확인
- ZIP 안에 `subaruu.conf`가 포함되지 않는 것 확인
- GitHub release asset digest와 로컬 SHA256 일치 확인
