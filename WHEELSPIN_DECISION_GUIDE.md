## 한 줄 결론

```text
수집 속도 = 라부엘토작
CR 안정성 = 마즈다작
초기 자금 부담 = 스바루작
```

조금 더 풀면 아래처럼 봅니다.

| 목적 | 추천 |
| --- | --- |
| Legendary Car / 휠스핀 전용차를 빨리 먹고 싶다 | 라부엘토작 |
| 돈을 안정적으로 벌거나 덜 잃고 싶다 | 마즈다작 |
| 시작 자금이 적다 | 스바루작 |
| 포인트당 슈퍼휠만 본다 | 마즈다작 |
| 시간당 수집차를 본다 | 라부엘토작 |

## 비교 대상

| 방식 | 1사이클 차량 수 | 보상 | 구매비 | 시간 |
| --- | ---: | --- | ---: | ---: |
| 라부엘토작 | 25대 | 슈퍼휠 25 + 일반휠 75 | 8,668,750 CR | 1:13:12 |
| 스바루작 | 33대 | 슈퍼휠 33 | 2,696,100 CR | 약 1:19:03 |
| 마즈다작 | 47대 | 슈퍼휠 47 | 4,465,000 CR | 1:29:17 |

슬롯으로 바꾸면 더 쉽습니다.

```text
슈퍼휠 1개 = 슬롯 3칸
일반휠 1개 = 슬롯 1칸
```

| 방식 | 슈퍼 슬롯 | 일반 슬롯 | 총 슬롯 |
| --- | ---: | ---: | ---: |
| 라부엘토작 | 75 | 75 | 150 |
| 스바루작 | 99 | 0 | 99 |
| 마즈다작 | 141 | 0 | 141 |

라부엘토작은 마즈다작보다 슈퍼휠 수는 적습니다.
대신 일반휠 75개를 같이 받고, 작업 시간이 더 짧습니다.

## 핵심 질문

중요한 질문은 하나입니다.

```text
일반휠 1칸이 슈퍼휠 1칸보다 얼마나 구린가?
```

기호로 쓰면:

```text
r = 일반 슬롯 가치 / 슈퍼 슬롯 가치
```

라부엘토작이 마즈다작보다 시간당 수집 효율에서 이기려면:

```text
r > 0.541
```

즉 일반휠이 슈퍼휠의 54.1% 가치만 있어도 라부엘토작이 시간당으로 앞섭니다.

## 내 수동 기록 결과

현재 개인 수동 기록은 아래와 같습니다.

### Legendary Car 기준

| 구분 | 슬롯 | Legendary Car | 확률 |
| --- | ---: | ---: | ---: |
| 일반 슬롯 | 808 | 13 | 1.609% |
| 슈퍼 슬롯 | 1,068 | 18 | 1.685% |

```text
r_LC = 1.609 / 1.685 = 0.955
```

### 휠스핀 전용차 기준

| 구분 | 슬롯 | WX | 확률 |
| --- | ---: | ---: | ---: |
| 일반 슬롯 | 808 | 9 | 1.114% |
| 슈퍼 슬롯 | 1,068 | 13 | 1.217% |

```text
r_WX = 1.114 / 1.217 = 0.915
```

둘 다 기준선보다 높습니다.

```text
라부엘토가 마즈다보다 시간당으로 이기는 조건 = r > 0.541
내 LC 기록 = 0.955
내 WX 기록 = 0.915
```

따라서 현재 수동 기록 기준으로는 라부엘토작이 시간당 수집 효율에서 앞섭니다.

## 외부 11,489 슬롯 표본이 왜 중요하냐

외부 표본은 일반휠과 슈퍼휠을 나눠 기록한 표본이 아닙니다.
그래서 일반휠 vs 슈퍼휠 비교에는 직접 쓰면 안 됩니다.

하지만 전체 휠스핀에서 Legendary Car가 어느 정도 나오는지는 아주 잘 보여줍니다.

외부 대형 표본:

| 항목 | 값 |
| --- | ---: |
| 전체 슬롯 | 11,489 |
| Legendary Car | 182, 1.584% |
| FE | 182, 1.584% |
| 비-FE WX | 157, 1.367% |
| WX + FE 전체 | 339, 2.951% |

이 값이 중요한 이유는 내 수동 기록과 거의 맞기 때문입니다.

```text
외부 LC = 1.584%
내 일반 LC = 1.609%
내 슈퍼 LC = 1.685%
```

즉 내 일반휠 기록이 이상하게 잘 나온 것도 아니고, 내 슈퍼휠 기록이 이상하게 못 나온 것도 아닙니다.
둘 다 외부 대형 표본의 LC 확률 근처에 있습니다.

그래서 이렇게 말할 수 있습니다.

```text
외부 표본은 일반 vs 슈퍼를 직접 비교하지는 못한다.
하지만 내 수동 기록이 정상 범위에 있다는 강한 검산 자료가 된다.
```

## 시간당 수집 효율

일반 슬롯과 슈퍼 슬롯이 거의 비슷하다고 보면, 결국 시간당 슬롯 수 싸움입니다.

| 방식 | 총 목표 슬롯 | 시간 | 시간당 목표 슬롯 |
| --- | ---: | ---: | ---: |
| 라부엘토작 | 150 | 1:13:12 | 123.0 |
| 마즈다작 | 141 | 1:29:17 | 94.8 |
| 스바루작 | 99 | 약 1:19:03 | 75.1 |

시간당 목표 슬롯은 라부엘토작이 가장 높습니다.

```text
라부엘토작 > 마즈다작 > 스바루작
```

## 포인트 효율

포인트만 보면 마즈다작이 강해 보입니다.

```text
마즈다작 = 21포인트로 슈퍼휠 1개
라부엘토작 = 39포인트로 슈퍼휠 1개 + 일반휠 3개
스바루작 = 30포인트로 슈퍼휠 1개
```

라부엘토작이 마즈다작보다 포인트당 수집 효율에서 이기려면:

```text
r > 0.857
```

현재 기록은:

```text
LC r = 0.955
WX r = 0.915
```

둘 다 0.857보다 높습니다.

따라서 현재 데이터 기준으로는 포인트당 수집 효율도 라부엘토작이 마즈다작보다 약간 좋을 가능성이 있습니다.
다만 이 차이는 크지 않습니다.

```text
포인트당 슈퍼휠만 보면 마즈다작
포인트당 수집차까지 보면 라부엘토작이 약간 우위 가능
```

## CR 효율

수집 효율과 CR 안정성은 다릅니다.

현재 개인 누적 CR 기준:

| 방식 | 실제 지갑 CR 손익 | 가치 포함 손익 |
| --- | ---: | ---: |
| 라부엘토작 | 약 +95만 / 사이클 | 약 +201만 / 사이클 |
| 스바루작 | 약 +361만 / 사이클 | 약 +419만 / 사이클 |
| 마즈다작 | 약 +452만 / 사이클 | 약 +535만 / 사이클 |

CR 안정성은 마즈다작이 가장 좋습니다.

```text
CR 안정성 = 마즈다작 > 스바루작 > 라부엘토작
```

라부엘토작은 좋은 차를 보관하면 지갑 CR이 흔들릴 수 있습니다.
따라서 라부엘토작은 CR 증식용이라기보다, CR 여유가 있을 때 수집 속도를 올리는 방식입니다.

```text
그래서 시간당 수집 효율은 라부엘토작이 가장 높다.
```
