# 휠스핀 작업 효율 메모

작성일: 2026-07-02

이 문서는 `스바루작`, `라부엘토작`, `마즈다작` 중 어떤 작업이 유리한지 비교하기 위한 연구 노트입니다.
아직 확정 결론이 아니라, 현재 실측값으로 세운 가설과 추가 표본 수집 기준을 정리한 문서입니다.

## 요약

- `효율`은 기준을 나누어 봐야 합니다.
- 포인트당 슈퍼휠, 지갑 CR, 초기 자금 부담 기준으로는 `마즈다작`이 유리합니다.
- CR이라는 제약을 제외하고 휠스핀 차량 획득 자체만 보면 `라부엘토작`이 더 최적화된 작업에 가깝습니다.
- 여기에 실제 소요 시간까지 반영하면 `라부엘토작`의 우위는 더 커집니다.
- 이 차이는 휠스핀 보상 테이블이 작업마다 달라진다는 뜻이 아닙니다.
- 핵심은 `라부엘토작`이 `마즈다작`보다 짧은 시간에 일반 휠스핀 슬롯까지 같이 가져온다는 점입니다.
- 현재 표본은 방향성 확인용입니다. 96% 수준으로 더 강하게 말하려면 일반 슬롯과 슈퍼 슬롯을 각각 약 1,000칸 이상 기록하는 것을 목표로 둡니다.

짧게 말하면 현재 가설은 아래처럼 분리합니다.

```text
돈/포인트 효율 = 마즈다작
CR 제약 없는 휠스핀 차량 획득 = 라부엘토작
시간당 희귀 휠스핀 차량 획득 = 라부엘토작
```

따라서 이 문서는 `라부엘토작이 모든 기준에서 무조건 좋다`는 추천글이 아닙니다.
돈과 포인트 효율은 마즈다작이 강하고, CR 여유가 있는 계정에서 휠스핀 차량을 노릴 때는 라부엘토작이 더 맞는 구조라는 가설을 검증하기 위한 표본 기록 노트입니다.

## 비교 대상

| 방식 | 차량 가격 | 사이클 대수 | 구매비 | 보상 |
| --- | ---: | ---: | ---: | --- |
| 라부엘토작 | 346,750 CR | 25대 | 8,668,750 CR | 슈퍼휠 25 + 일반휠 75 |
| 스바루작 | 81,700 CR | 33대 | 2,696,100 CR | 슈퍼휠 33 |
| 마즈다작 | 95,000 CR | 47대 | 4,465,000 CR | 슈퍼휠 47 |

슬롯 기준:

| 방식 | 슈퍼 슬롯 | 일반 슬롯 |
| --- | ---: | ---: |
| 라부엘토작 | 75 | 75 |
| 스바루작 | 99 | 0 |
| 마즈다작 | 141 | 0 |

## 변수

```text
S = 슈퍼 슬롯 1칸의 Legendary Car 기대 가치
W = 일반 슬롯 1칸의 Legendary Car 기대 가치
r = W / S
```

`r`은 일반 슬롯이 슈퍼 슬롯 대비 어느 정도 가치인지 나타냅니다.

## 시간 실측

현재 로그 기준 비교:

| 방식 | 실측 시간 |
| --- | ---: |
| 라부엘토작 | 1시간 13분 12초, 4,392초 |
| 마즈다작 | 1시간 29분 17초, 5,357초 |

시간비:

```text
T_R / T_M = 4392 / 5357 = 0.8199
```

즉 현재 실측에서는 라부엘토작이 마즈다작 시간의 약 82.0%이고, 약 18.0% 빠릅니다.

## 라부엘토작 vs 마즈다작

시간이 같다면 라부엘토작이 마즈다작보다 보상량 기준으로 이기려면 일반 슬롯 가치가 슈퍼 슬롯의 약 88% 이상이어야 합니다.
하지만 현재 비교의 핵심은 시간이 같지 않다는 점입니다.
현재 로그에서는 라부엘토작이 마즈다작보다 약 18% 빠릅니다.

라부엘토작 시간당 기대값:

```text
(75S + 75W) / T_R
```

마즈다작 시간당 기대값:

```text
141S / T_M
```

라부엘토작이 시간당 Legendary Car 기대값에서 이기려면:

```text
75S(1+r) / T_R > 141S / T_M
r > (141 / 75) * (T_R / T_M) - 1
r > 1.88 * 0.8199 - 1
r > 0.541
```

따라서 현재 시간 실측이 유지된다면, 일반 슬롯의 Legendary Car 가치가 슈퍼 슬롯의 약 54.1%를 넘을 때 라부엘토작이 시간당 기준으로 마즈다작보다 유리합니다.

공개적으로 설명할 때는 `88%`보다 `54.1%` 기준을 먼저 말하는 편이 덜 헷갈립니다.
`88%`는 시간이 같을 때의 기준이고, `54.1%`는 현재 실측 시간 차이를 반영한 기준입니다.

## 현재 표본

현재 기록된 표본:

| 구분 | 슬롯 수 | Legendary Car | 관측 확률 |
| --- | ---: | ---: | ---: |
| 일반 슬롯 | 89 | 2 | 2.25% |
| 슈퍼 슬롯 | 234 | 4 | 1.71% |

현재 표본의 비율:

```text
r = (2 / 89) / (4 / 234) = 1.315
```

이 표본만 보면 `1.315 > 0.541`이므로 라부엘토작 우세 쪽을 지지합니다.
하지만 LC 개수가 각각 2개, 4개뿐이라 확률 확정용으로는 부족합니다.

현재 표본은 `일반 슬롯이 슈퍼 슬롯보다 확실히 좋다`고 말하기보다는, `일반 슬롯 확률이 슈퍼 슬롯과 거의 동등하거나 최소한 크게 밀리지 않는다`는 쪽에 가깝게 해석합니다.
라부엘토작은 일반 슬롯이 슈퍼 슬롯과 완전히 같을 필요도 없습니다.
현재 시간 실측 기준에서는 일반 슬롯 가치가 슈퍼 슬롯의 54.1%만 넘어도 시간당 Legendary Car 기대값에서 마즈다작을 앞설 수 있습니다.

## 필요한 추가 표본

96% 수준에서 더 강하게 비교하려면 휠 개수보다 슬롯 개수를 기준으로 기록하는 편이 좋습니다.

권장 목표:

| 목표 | 일반 슬롯 | 슈퍼 슬롯 | 실제 휠 기준 |
| --- | ---: | ---: | --- |
| 최소 판정 시도 | 600 | 600 | 일반휠 600개 + 슈퍼휠 200개 |
| 권장 | 1,000 | 1,000 | 일반휠 1,000개 + 슈퍼휠 334개 |
| 더 안정적 | 1,500 | 1,500 | 일반휠 1,500개 + 슈퍼휠 500개 |

현재 표본을 포함하면 권장 목표까지 대략 아래 정도가 더 필요합니다.

| 목표 | 추가 일반휠 | 추가 슈퍼휠 |
| --- | ---: | ---: |
| 600 슬롯씩 | 약 511개 | 약 122개 |
| 1,000 슬롯씩 | 약 911개 | 약 256개 |

라부엘토작 1사이클은 일반 슬롯 75칸과 슈퍼 슬롯 75칸을 같이 만듭니다.
따라서 현재 표본 포함 기준으로는 라부엘토작을 약 13사이클 더 돌리면 일반/슈퍼 각각 1,000슬롯 근처까지 갈 수 있습니다.

## 기록 양식

아래 네 가지만 기록하면 됩니다.

```text
일반휠:
- 일반휠 개수
- 일반 슬롯 수
- Legendary Car 수

슈퍼휠:
- 슈퍼휠 개수
- 슈퍼 슬롯 수
- Legendary Car 수
```

슈퍼휠 1개는 슈퍼 슬롯 3칸으로 계산합니다.

## CR 관점

현재 실측 CR 기준으로는 마즈다작이 더 안정적입니다.

| 방식 | 실제 지갑 CR 손익 | 보유 차량 가치 포함 손익 |
| --- | ---: | ---: |
| 라부엘토작 | 약 -154만 CR / 사이클 | 약 +213만 CR / 사이클 |
| 마즈다작 | 약 +302만 CR / 사이클 | 약 +441만 CR / 사이클 |

따라서 라부엘토작은 수집 기대값 쪽 가설이고, CR 증식용 결론은 아닙니다.
초기 자금이 부족하거나 지갑 CR 감소가 부담되면 마즈다작이 더 안전합니다.

단, 라부엘토작의 `약 -154만 CR / 사이클`은 1사이클만 따져본 지갑 CR 표본입니다.
해당 표본에는 Lamborghini Sesto가 뜬 변수가 섞여 있으므로, 장기 평균 손익으로 확정한 값은 아닙니다.

또한 포인트당 슈퍼휠만 보면 마즈다작이 좋아 보이는 것이 정상입니다.
마즈다작은 21포인트로 슈퍼휠 1개를 얻고, 라부엘토작은 약 39~40포인트로 슈퍼휠 1개와 일반휠 3개를 얻는 구조입니다.
이 기준에서는 마즈다작이 강합니다.
이 문서에서 확인하려는 것은 포인트당 슈퍼휠이 아니라, 실제 소요 시간을 반영한 시간당 희귀 휠스핀 차량 기대값입니다.

## 커뮤니티 반론에 대한 정리

`마즈다작을 돌렸더니 휠스핀 차량을 많이 먹었다`는 체감은 자연스럽습니다.
마즈다작은 한 사이클에 슈퍼 슬롯 141칸을 제공하므로 사이클당 휠스핀 차량 체감이 강할 수 있습니다.

다만 이 체감만으로 시간당 효율 비교가 끝나지는 않습니다.
비교해야 하는 값은 아래입니다.

```text
같은 시간 동안
몇 슬롯을 돌렸고
Legendary Car 또는 목표 차량이 몇 개 나왔는가
```

라부엘토작 가설은 `라부엘토작의 휠스핀 보상 테이블이 더 좋다`가 아닙니다.
현재 가설은 `라부엘토작이 더 빠르고, 일반 휠스핀 슬롯까지 함께 가져오기 때문에 시간당 목표 차량 기대값이 높을 수 있다`입니다.

따라서 반론도 기준을 나누어 해석해야 합니다.

| 반론 | 판단 |
| --- | --- |
| 마즈다작이 돈이 덜 든다 | 맞음 |
| 마즈다작이 포인트당 슈퍼휠이 좋다 | 맞음 |
| 마즈다작을 돌리면 휠스핀 차량을 많이 먹는 체감이 있다 | 자연스러움 |
| 그래서 시간당 희귀 차량 기대값도 반드시 마즈다가 높다 | 아직 별도 검증 필요 |

현재 필요한 것은 선호도 논쟁이 아니라 같은 기준으로 쌓은 표본입니다.
일반 슬롯과 슈퍼 슬롯에서 목표 차량이 얼마나 나오는지 기록해야 시간당 비교가 가능합니다.

## 현재 운영 결론

- 돈이 부족하면 `마즈다작`
- CR을 안정적으로 늘리고 싶으면 `마즈다작`
- 포인트당 슈퍼휠을 중시하면 `마즈다작`
- CR이라는 요소를 제외하고 휠스핀 차량 획득 자체를 노리면 `라부엘토작`
- 시간까지 반영해서 희귀 휠스핀 차량을 최대한 빨리 먹는 목적이면 `라부엘토작`이 더 유리한 쪽
- 최종 확정은 일반/슈퍼 각각 1,000슬롯 이상 추가 기록 뒤 다시 판단

현재 운영 가설은 `마즈다작 = 돈/포인트 안정형`, `라부엘토작 = CR 여유가 있을 때 휠스핀 차량 수집 최적화형`입니다.
확률 자체는 거의 동등하다고 보수적으로 놓더라도, 시간이라는 요소를 추가하면 라부엘토작 쪽이 더 강하게 유리해집니다.
