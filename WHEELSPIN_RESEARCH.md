# 휠스핀 작업 효율 메모

작성일: 2026-07-02  
갱신: 2026-07-04

이 문서는 `스바루작`, `라부엘토작`, `마즈다작` 중 어떤 작업이 유리한지 비교하기 위한 연구 노트입니다.
아직 확정 결론이 아니라, 현재 실측값으로 세운 가설과 추가 표본 수집 기준을 정리한 문서입니다.

원본 기록은 [휠스핀 실측 원본 기록](research/wheelspin-raw-20260702.md)에 따로 보존합니다.
외부 대형 표본은 [외부 11,489-slot 표본 분석](research/external-11489/ANALYSIS.md)에 따로 분리했습니다.

## 요약

- `효율`은 기준을 나누어 봐야 합니다.
- 포인트당 슈퍼휠, 지갑 CR, 초기 자금 부담 기준으로는 `마즈다작`이 유리합니다.
- 초기 구매비만 낮게 잡으면 `스바루작`도 저부담 기준선으로 볼 수 있습니다.
- CR 제약이 없고 실제 소요 시간까지 반영하면 `라부엘토작`이 Legendary Car 획득에 더 최적화된 작업에 가깝습니다.
- 이 차이는 휠스핀 보상 테이블이 작업마다 달라진다는 뜻이 아닙니다.
- 핵심은 `라부엘토작`이 `마즈다작`보다 짧은 시간에 일반 휠스핀 슬롯까지 같이 가져온다는 점입니다.
- 현재 개인 실측 표본에서는 Legendary Car 기준 `r = 0.955`, 휠스핀 전용차 기준 `r_WX = 0.915`로 관측되었습니다. 둘 다 시간당 기준선인 `r > 0.541`을 넘습니다.
- 외부 11,489-slot 표본에서는 LC가 약 `1.584%`, FE가 약 `1.584%`, 비-FE WX가 약 `1.340%`로 관측되었습니다. 이 값은 참고 기준값으로만 사용합니다.

짧게 말하면 현재 가설은 아래처럼 분리합니다.

```text
돈/포인트 효율 = 마즈다작
저자본/저부담 슈퍼휠 기준선 = 스바루작
CR 제약 없고 시간까지 반영한 Legendary Car 획득 = 라부엘토작
시간당 Legendary Car 획득 = 라부엘토작
```

이 문서는 `라부엘토작이 모든 기준에서 무조건 좋다`는 추천글이 아닙니다.
돈과 포인트 효율은 마즈다작이 강하고, CR 여유가 있는 계정에서 Legendary Car 또는 휠스핀 전용차를 노릴 때는 라부엘토작이 더 맞는 구조라는 가설을 검증하기 위한 표본 기록 노트입니다.

## 비교 대상

| 방식 | 차량 가격 | 사이클 대수 | 구매비 | 보상 |
| --- | ---: | ---: | ---: | --- |
| 라부엘토작 | 346,750 CR | 25대 | 8,668,750 CR | 슈퍼휠 25 + 일반휠 75 |
| 스바루작 | 81,700 CR | 33대 | 2,696,100 CR | 슈퍼휠 33 |
| 마즈다작 | 95,000 CR | 47대 | 4,465,000 CR | 슈퍼휠 47 |

슬롯 기준:

| 방식 | 슈퍼 슬롯 | 일반 슬롯 |
| --- | ---: | ---: |
| 라부엘토작 | 75 | 75 |
| 스바루작 | 99 | 0 |
| 마즈다작 | 141 | 0 |

스바루작은 비교표에는 남기지만, 본론은 라부엘토작과 마즈다작 비교에 집중합니다.
스바루작은 차량 가격이 낮다는 장점이 있지만, 포인트당 슈퍼휠과 슬롯 수 기준에서는 마즈다작 쪽이 더 강합니다.

## 변수

```text
S = 슈퍼 슬롯 1칸의 Legendary Car 기대 가치
W = 일반 슬롯 1칸의 Legendary Car 기대 가치
r = W / S
```

`r`은 일반 슬롯이 슈퍼 슬롯 대비 어느 정도 가치인지 나타냅니다.

## 목표 차량 범위

이 문서의 핵심 수식에서 목표 차량은 우선 `Legendary Car`로 정의합니다.
FE 차량은 별도로 집계하고, 휠스핀 전용차는 `WX`로 따로 집계합니다.

외부 표본 분석에서는 아래처럼 구분합니다.

```text
WX_source = Wheelspin / Seasonal 보상 차량 45종 전체
WX_non_FE = WX_source 중 FE를 제외한 차량
FE = Forza Edition 차량
```

개인 실측의 `WX`는 실전 수집 가치 기준이므로, FE와 분리해서 해석합니다.

## 시간 실측

현재 로그 기준 비교:

| 방식 | 실측 시간 |
| --- | ---: |
| 라부엘토작 | 1시간 13분 12초, 4,392초 |
| 마즈다작 | 1시간 29분 17초, 5,357초 |

시간비:

```text
T_R / T_M = 4392 / 5357 = 0.8199
```

즉 현재 실측에서는 라부엘토작이 마즈다작 시간의 약 82.0%이고, 약 18.0% 빠릅니다.

## 스바루작 시간 추정

스바루작은 직접 시간 로그가 없으므로, 라부엘토작과 마즈다작의 실측 시간을 차량 대수 기준으로 선형 보간했습니다.

```text
라부엘토작 = 25대, 4,392초
마즈다작 = 47대, 5,357초

대수 차이 = 22대
시간 차이 = 965초
차량 1대당 추가 시간 = 965 / 22 = 43.86초
```

스바루작은 33대이므로 라부엘토작보다 8대 많습니다.

```text
스바루작 추정 시간 = 4,392 + 8 * 43.86
= 4,743초
= 약 1시간 19분 03초
```

스바루작은 33대 기준 슈퍼휠 33개, 즉 슈퍼 슬롯 99칸을 제공합니다.

```text
스바루작 시간당 슈퍼 슬롯 = 99 / 4743 * 3600 = 75.1칸/시간
마즈다작 시간당 슈퍼 슬롯 = 141 / 5357 * 3600 = 94.8칸/시간
```

따라서 마즈다작은 스바루작보다 시간당 슈퍼 슬롯을 약 26% 더 많이 제공합니다.

## 라부엘토작 vs 마즈다작

시간이 같다면 라부엘토작이 마즈다작보다 보상량 기준으로 이기려면 일반 슬롯 가치가 슈퍼 슬롯의 약 88% 이상이어야 합니다.
하지만 현재 비교의 핵심은 시간이 같지 않다는 점입니다.
현재 로그에서는 라부엘토작이 마즈다작보다 약 18% 빠릅니다.

라부엘토작 시간당 기대값:

```text
(75S + 75W) / T_R
```

마즈다작 시간당 기대값:

```text
141S / T_M
```

라부엘토작이 시간당 Legendary Car 기대값에서 이기려면:

```text
75S(1+r) / T_R > 141S / T_M
r > (141 / 75) * (T_R / T_M) - 1
r > 1.88 * 0.8199 - 1
r > 0.541
```

따라서 현재 시간 실측이 유지된다면, 일반 슬롯의 Legendary Car 가치가 슈퍼 슬롯의 약 54.1%를 넘을 때 라부엘토작이 시간당 기준으로 마즈다작보다 유리합니다.

`88%`는 시간이 같을 때의 기준이고, `54.1%`는 현재 실측 시간 차이를 반영한 기준입니다.

## 라부엘토작 vs 스바루작

스바루작은 슈퍼 슬롯만 제공하므로 기대값은 아래처럼 단순화할 수 있습니다.

```text
스바루작 = 99S
라부엘토작 = 75S + 75W = 75S(1+r)
```

보상량 기준으로 라부엘토작이 스바루작보다 앞서려면:

```text
75S + 75W > 99S
75W > 24S
W/S > 0.32
```

시간 추정까지 넣으면 기준은 더 낮아집니다.

```text
라부엘토작 = 75S(1+r) / 4392
스바루작 = 99S / 4743
```

라부엘토작이 시간당 기준으로 스바루작보다 앞서려면:

```text
75S(1+r) / 4392 > 99S / 4743
r > 0.222
```

따라서 스바루작은 저자본 기준선으로 의미가 있지만, Legendary Car 시간당 기대값 비교에서는 라부엘토작보다 불리한 쪽입니다.

## 현재 개인 누적 표본

현재까지 수기로 파싱 가능한 개인 표본을 합치면 아래와 같습니다.
개별 표본의 원자료는 별도 기록에 보존하고, 본문에서는 누적값만 사용합니다.

| 구분 | 슬롯 수 | Legendary Car | 관측 확률 |
| --- | ---: | ---: | ---: |
| 일반 슬롯 | 808 | 13 | 1.609% |
| 슈퍼 슬롯 | 1,068 | 18 | 1.685% |

누적 비율:

```text
W_total = 13 / 808 = 1.609%
S_total = 18 / 1068 = 1.685%
r_total = W_total / S_total = 0.955
```

```text
0.955 > 0.541
```

따라서 현재까지의 개인 누적 표본은 라부엘토작이 시간당 Legendary Car 기대값에서 마즈다작보다 유리하다는 가설을 지지합니다.
다만 누적 Legendary Car 개수는 일반 13개, 슈퍼 18개이므로 아직 확률 확정용 표본은 아닙니다.

## 휠스핀 전용차 보조 통계

핵심 수식은 Legendary Car 기준으로 계산하지만, 휠스핀 전용차도 별도 보조 통계로 기록합니다.

```text
WX = Wheelspin Exclusive Car
```

WX는 명시적으로 휠스핀 전용차라고 기록한 차량과, 원본 기록에서 이후 휠스핀 전용차로 확인한 보관 차량을 포함합니다.
LC이면서 동시에 WX인 차량도 WX에 포함합니다.
FE 차량은 WX에 넣지 않습니다.

일부 휠스핀 전용차는 차량 목록을 확인하기 전에 판매됐을 가능성이 있습니다.
따라서 현재 WX 집계는 보수적 하한값입니다.
실제 WX 획득 수는 현재 기록값보다 높을 수 있습니다.

현재 누적 WX 표본은 아래와 같습니다.

| 구분 | 슬롯 수 | WX | 관측 확률 |
| --- | ---: | ---: | ---: |
| 일반 슬롯 | 808 | 9 | 1.114% |
| 슈퍼 슬롯 | 1,068 | 13 | 1.217% |

```text
N_WX = 9 / 808 = 1.114%
S_WX = 13 / 1068 = 1.217%
r_WX = N_WX / S_WX = 0.915
```

현재 표본에서는 슈퍼 슬롯의 WX 관측률이 약간 높게 나왔습니다.
다만 WX 개수는 일반 9개, 슈퍼 13개뿐이므로 확률 확정용으로는 부족합니다.
그리고 이 값은 보수적 하한값입니다.

현재 관측값을 작업별로 환산하면 아래와 같습니다.

| 방식 | WX 기대값 / 사이클 |
| --- | ---: |
| 라부엘토작 | 약 1.75 |
| 마즈다작 | 약 1.72 |
| 스바루작 | 약 1.21 |

시간당 추정값은 아래와 같습니다.

| 방식 | WX / 시간 |
| --- | ---: |
| 라부엘토작 | 약 1.43 |
| 마즈다작 | 약 1.15 |
| 스바루작 | 약 0.91 |

현재 WX 표본도 라부엘토작의 수집 효율 가설을 지지합니다.
다만 표본 수가 적고 누락 가능성이 있기 때문에, Legendary Car 통계보다 더 보수적으로 해석해야 합니다.

## 외부 11,489-slot 표본 참고 기준

외부 대형 표본은 `일반휠`과 `슈퍼휠`을 분리하지 않은 혼합 슬롯 표본입니다.
따라서 이 표본은 `W/S` 계산에는 사용하지 않고, 전체 휠스핀 보상 분포의 참고 기준값으로만 사용합니다.

외부 표본의 핵심값은 아래와 같습니다.

| 항목 | 값 |
| --- | ---: |
| 전체 슬롯 | 11,489 |
| Legendary Car | 182, 1.584% |
| FE | 182, 1.584% |
| WX_non_FE | 154, 1.340% |
| WX_source 전체 | 336, 2.925% |
| CR 보상 비율 | 76.2% |
| 차량 보상 비율 | 23.8% |
| 모든 차량 즉시판매 평균 | 94,401 CR / slot |
| Legendary/FE 보유 평균 | 51,395 CR / slot |
| CR only 평균 | 39,790 CR / slot |

개인 실측의 LC 관측률은 외부 대형 표본의 LC 확률과 가까운 범위에 있습니다.

```text
외부 표본 LC = 182 / 11,489 = 1.584%
개인 실측 일반 LC = 13 / 808 = 1.609%
개인 실측 슈퍼 LC = 18 / 1,068 = 1.685%
```

이 비교는 개인 실측의 확률 규모가 비정상적으로 보이지 않는다는 참고 근거입니다.
다만 외부 표본은 일반/슈퍼 분리 표본이 아니므로, 개인 실측 표본과 단순 합산하지 않습니다.

자세한 내용은 [외부 11,489-slot 표본 분석](research/external-11489/ANALYSIS.md)을 봅니다.

## 표본 해석

현재 표본은 `일반 슬롯이 슈퍼 슬롯보다 확실히 좋다`고 말하기보다는, `일반 슬롯 확률이 슈퍼 슬롯과 거의 동등하거나 최소한 크게 밀리지 않는다`는 쪽에 가깝게 해석합니다.
라부엘토작은 일반 슬롯이 슈퍼 슬롯과 완전히 같을 필요도 없습니다.
현재 시간 실측 기준에서는 일반 슬롯 가치가 슈퍼 슬롯의 54.1%만 넘어도 시간당 Legendary Car 기대값에서 마즈다작을 앞설 수 있습니다.

## 필요한 추가 표본

더 강하게 비교하려면 휠 개수보다 슬롯 개수를 기준으로 기록하는 편이 좋습니다.
아래 목표치는 확정 판정선이 아니라 현재 표본보다 더 안정적인 1차 검증 목표입니다.
실제 신뢰도는 최종적으로 관측된 Legendary Car 개수에 따라 다시 계산해야 합니다.

권장 목표:

| 목표 | 일반 슬롯 | 슈퍼 슬롯 | 실제 휠 기준 |
| --- | ---: | ---: | --- |
| 최소 판정 시도 | 600 | 600 | 일반휠 600개 + 슈퍼휠 200개 |
| 권장 | 1,000 | 1,000 | 일반휠 1,000개 + 슈퍼휠 334개 |
| 더 안정적 | 1,500 | 1,500 | 일반휠 1,500개 + 슈퍼휠 500개 |

현재 누적 표본을 포함하면 목표까지 대략 아래 정도가 더 필요합니다.

| 목표 | 추가 일반휠 | 추가 슈퍼휠 |
| --- | ---: | ---: |
| 600 슬롯씩 | 0개 | 0개 |
| 1,000 슬롯씩 | 약 192개 | 0개 |
| 1,500 슬롯씩 | 약 692개 | 약 144개 |

라부엘토작 1사이클은 일반 슬롯 75칸과 슈퍼 슬롯 75칸을 같이 만듭니다.
따라서 라부엘토작을 추가로 돌리면 일반/슈퍼 표본이 같은 속도로 쌓입니다.

## 기록 양식

아래 값만 기록하면 됩니다.

```text
일반휠:
- 일반휠 개수
- 일반 슬롯 수
- Legendary Car 수
- WX 수

슈퍼휠:
- 슈퍼휠 개수
- 슈퍼 슬롯 수
- Legendary Car 수
- WX 수
```

슈퍼휠 1개는 슈퍼 슬롯 3칸으로 계산합니다.

CR 손익까지 보려면 아래 값도 같이 기록합니다.

```text
- 시작 CR
- 종료 CR
- 팔지 않고 보유한 차량의 즉시판매 상당액 합계
```

CR 색깔, 일반 차량 등급은 이 문서의 핵심 수식에는 필요하지 않습니다.
FE 차량은 WX와 분리해서 기록합니다.

## CR 관점

CR 관점에서는 표본 편차가 큽니다.
따라서 아래 값은 장기 평균 확정값이 아니라, 현재 누적 표본을 각 작업 보상 구조에 환산한 값입니다.

슈퍼휠 쪽에는 35,000,000 CR 차량 1대가 포함되어 있습니다.
이 값은 실제 상방 사례로 별도 기록하되, 아래 평균에서는 제외한 보정값을 사용합니다.

| 구분 | 실제 CR 평균 | 보유 차량 가치 포함 평균 |
| --- | ---: | ---: |
| 일반휠 | 64,582 CR / 개 | 72,852 CR / 개 |
| 슈퍼휠 | 191,123 CR / 개 | 208,742 CR / 개 |

누적 평균을 각 작업에 환산하면 아래와 같습니다.

| 방식 | 실제 지갑 CR 손익 | 보유 차량 가치 포함 손익 |
| --- | ---: | ---: |
| 라부엘토작 | 약 +95만 CR / 사이클 | 약 +201만 CR / 사이클 |
| 스바루작 | 약 +361만 CR / 사이클 | 약 +419만 CR / 사이클 |
| 마즈다작 | 약 +452만 CR / 사이클 | 약 +535만 CR / 사이클 |

현재 누적 표본 기준으로는 라부엘토작도 실제 지갑 CR 기준 흑자권입니다.
다만 CR 안정성 기준에서는 여전히 스바루작과 마즈다작이 더 안전합니다.
스바루작은 선지출이 가장 낮고, 마즈다작은 포인트당 슈퍼휠과 시간당 슈퍼 슬롯이 더 좋습니다.

따라서 라부엘토작은 CR 증식용 결론이 아니라, CR 여유가 있을 때 시간당 Legendary Car 수집을 노리는 쪽의 가설입니다.
초기 자금이 부족하거나 지갑 CR 감소가 부담되면 스바루작 또는 마즈다작이 더 안전합니다.

또한 포인트당 슈퍼휠만 보면 마즈다작이 좋아 보이는 것이 정상입니다.
마즈다작은 21포인트로 슈퍼휠 1개를 얻고, 라부엘토작은 39포인트로 슈퍼휠 1개와 일반휠 3개를 얻는 구조입니다.
포인트당 슈퍼휠만 보면 39포인트 기준에서도 마즈다작이 강합니다.
다만 일반휠 3개의 Legendary Car 가치를 함께 넣으면, 순수 포인트 효율 비교는 `r` 값에 따라 갈립니다.
이 문서에서 확인하려는 것은 포인트당 슈퍼휠이 아니라, 실제 소요 시간을 반영한 시간당 Legendary Car 기대값입니다.

## 커뮤니티 반론에 대한 정리

`마즈다작을 돌렸더니 휠스핀 차량을 많이 먹었다`는 체감은 자연스럽습니다.
마즈다작은 한 사이클에 슈퍼 슬롯 141칸을 제공하므로 사이클당 휠스핀 차량 체감이 강할 수 있습니다.

스바루작도 슈퍼휠만 제공하므로 체감은 단순합니다.
다만 스바루작은 마즈다작보다 포인트당 슈퍼휠과 시간당 슈퍼 슬롯이 낮습니다.

다만 이 체감만으로 시간당 효율 비교가 끝나지는 않습니다.
비교해야 하는 값은 아래입니다.

```text
같은 시간 동안
몇 슬롯을 돌렸고
Legendary Car 또는 WX가 몇 개 나왔는가
```

라부엘토작 가설은 `라부엘토작의 휠스핀 보상 테이블이 더 좋다`가 아닙니다.
현재 가설은 `라부엘토작이 더 빠르고, 일반 휠스핀 슬롯까지 함께 가져오기 때문에 시간당 목표 차량 기대값이 높을 수 있다`입니다.

따라서 반론도 기준을 나누어 해석해야 합니다.

| 반론 | 판단 |
| --- | --- |
| 스바루작이 초기 비용이 가장 낮다 | 맞음 |
| 마즈다작이 돈이 덜 든다 | 라부엘토작과 비교하면 맞음 |
| 마즈다작이 포인트당 슈퍼휠이 좋다 | 맞음 |
| 마즈다작을 돌리면 휠스핀 차량을 많이 먹는 체감이 있다 | 자연스러움 |
| 그래서 시간당 Legendary Car 기대값도 반드시 마즈다가 높다 | 아직 별도 검증 필요 |

현재 필요한 것은 선호도 논쟁이 아니라 같은 기준으로 쌓은 표본입니다.
일반 슬롯과 슈퍼 슬롯에서 Legendary Car와 WX가 얼마나 나오는지 기록해야 시간당 비교가 가능합니다.

## 현재 운영 결론

- 초기 구매비를 가장 낮게 잡으면 `스바루작`
- 돈이 부족하면 `스바루작` 또는 `마즈다작`
- CR을 안정적으로 늘리고 싶으면 `마즈다작`
- 포인트당 슈퍼휠을 중시하면 `마즈다작`
- 시간당 슈퍼 슬롯을 중시하면 `마즈다작`
- CR 제약이 없고 일반휠 가치를 인정하면 `라부엘토작`도 비교 대상에 들어옴
- 시간까지 반영해서 Legendary Car 또는 WX를 최대한 빨리 먹는 목적이면 `라부엘토작`이 더 유리한 쪽
- 일반/슈퍼 각각 1,000슬롯 이상을 1차 검증 목표로 두고, 관측 LC/WX 개수에 따라 신뢰도를 다시 계산

현재 운영 가설은 `스바루작 = 저자본 기준선`, `마즈다작 = 돈/포인트 안정형`, `라부엘토작 = CR 여유가 있을 때 시간당 수집형`입니다.
확률 자체는 거의 동등하다고 보수적으로 놓더라도, 시간이라는 요소를 추가하면 라부엘토작 쪽이 더 강하게 유리해집니다.

이 글의 결론은 `라부엘토작이 모든 유저에게 최고`가 아닙니다.
스바루작은 초기 비용이 낮고, 마즈다작은 저자본, CR 안정성, 포인트당 슈퍼휠 기준에서 더 안전합니다.
다만 현재 실측 시간에서는 라부엘토작이 마즈다작보다 약 18% 빠르기 때문에, Legendary Car를 시간당 얼마나 얻느냐는 기준에서는 일반 슬롯 가치가 슈퍼 슬롯의 54.1%만 넘어도 라부엘토작이 앞섭니다.
현재까지의 LC/WX 누적 표본도 이 기준을 넘는 방향입니다.
따라서 논쟁의 핵심은 감상이 아니라 `r = W / S`를 추가 표본으로 검증하는 것입니다.
