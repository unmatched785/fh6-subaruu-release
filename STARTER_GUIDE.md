# 돈 없는 사람을 위한 스바루런 시작 루트

이 문서는 기능 설명서가 아닙니다.
처음 시작하는 사람이 "뭘 먼저 해야 하지?"에서 멈추지 않게 하려고 만든 시작 루트입니다.

`README.md`는 버튼, 탭, 설정값을 설명합니다.
이 문서는 그 전에 읽는 글입니다.

## 스바루런이란?

스바루런은 이벤트랩을 돌며 스킬 포인트를 얻고, 그 포인트로 특정 차량의 자동차 마스터리를 찍어 슈퍼휠스핀을 얻는 반복 작업입니다.

흐름은 단순합니다.

```text
이벤트랩 주행 -> 차량 구매 -> 자동차 마스터리 -> 슈퍼휠스핀 획득 -> 차량 삭제
```

하지만 처음부터 크게 돌리면 위험합니다.
차량을 살 돈이 부족하면 중간에 구매가 멈추고, 그 상태로 삭제 루프가 이어질 수 있습니다.
그래서 이 도구는 `보유CR`을 입력하게 하고, 가진 돈으로 가능한 범위까지만 돌리게 합니다.

핵심은 이것입니다.

```text
처음부터 33대 x 여러 회차를 돌리는 작업이 아니다.
가진 돈으로 살 수 있는 만큼만 시작해서, 번 돈으로 점점 키우는 작업이다.
```

## 초보자의 첫 목표

처음 목표는 "큰돈 벌기"가 아닙니다.
먼저 안전하게 한 번이라도 돌아가는 상태를 만드는 것입니다.

첫 목표:

1. 작업차 1대를 준비합니다.
2. 작업차에 필요한 자동차 마스터리 포인트를 확보합니다.
3. 가능하면 의상, 경적, 제스처 같은 휠스핀 잡보상을 먼저 사둡니다.
4. 보유 CR을 입력하고 `적용`을 눌러 가능한 구매 대수만 확인합니다.
5. 가능한 만큼만 돌려서 소량의 슈퍼휠스핀을 얻습니다.
6. 얻은 CR로 다음 실행 규모를 조금씩 늘립니다.

처음부터 "완전 자동으로 밤새 돌리기"를 목표로 잡으면 안 됩니다.
처음 목표는 "내 계정과 내 PC에서 작은 흐름이 안전하게 이어지는지 확인"입니다.

## 왜 의상, 경적, 제스처를 먼저 사나?

휠스핀은 CR만 주는 것이 아닙니다.
의상, 경적, 제스처 같은 보상도 나옵니다.

이 보상들이 많이 남아 있으면, 초반 휠스핀에서 기대한 만큼 CR이 안 나올 수 있습니다.
스바루런으로 돈을 불리려는 사람에게는 이게 꽤 아픕니다.

그래서 가능하면 먼저 아래 보상을 상점에서 사둡니다.

- 의상
- 경적
- 제스처
- 기타 휠스핀에서 자주 나오는 싼 장식 보상

이 작업은 "필수"라기보다는 손해를 줄이는 준비입니다.
돈이 너무 없으면 전부 사기 어렵습니다.
그 경우에는 일단 가능한 만큼만 사두고, 스바루런으로 돈을 조금 벌 때마다 다시 채워도 됩니다.

정확히 얼마가 필요한지는 계정 상태에 따라 달라질 수 있습니다.
이 문서는 아직 전체 구매 비용을 확정값으로 적지 않습니다.

## 첫 작업차 1대 만들기

처음에는 작업차를 많이 살 필요가 없습니다.
먼저 작업차 1대를 준비하고, 그 차로 이벤트랩 주행과 마스터리 흐름이 되는지 확인합니다.

스바루작 기준 작업차는 아래 차량입니다.

```text
1998 SUBARU IMPREZA 22B-STI VERSION
```

준비할 것:

- 작업차 1대
- 이벤트랩을 돌 수 있는 상태
- 작업차 마스터리를 찍을 수 있는 스킬 포인트
- 작업차 즐겨찾기
- 스바루작 이벤트랩 즐겨찾기 또는 맵 코드

처음에는 작업차 즐겨찾기를 가능하면 해당 1대만 남기는 편이 안전합니다.
즐겨찾기가 많거나 최근 추가됨 목록이 섞이면 선택 대상이 꼬일 수 있습니다.

## 보유 CR로 가능한 만큼만 돌리기

스바루작 기준 차량 가격은 현재 `81,700 CR`입니다.

예를 들면 이렇습니다.

| 보유 CR | 가능한 스바루 구매 대수 | 의미 |
| ---: | ---: | --- |
| 81,699 CR 이하 | 0대 | 실행 막힘 |
| 81,700 CR | 1대 | 차량 1대는 가능하지만 삭제할 여유가 거의 없음 |
| 163,400 CR | 2대 | 구매 2대, 강화 2대, 삭제 1대 같은 아주 작은 확인 가능 |
| 817,000 CR | 10대 | 1회차 안에서 10대 규모 가능 |
| 2,696,100 CR | 33대 | 스바루작 1사이클 기준 구매 33대 가능 |

정확한 값은 직접 계산하지 않아도 됩니다.
`올인원` 탭에서 현재 게임에 들고 있는 `보유CR`을 입력하고 `적용`을 누르면 됩니다.

`적용` 버튼은 실행 버튼이 아닙니다.
현재 보유 CR을 기준으로 프리셋 1회차 구성과 회차를 다시 계산하는 버튼입니다.
돈이 많다고 회차를 자동으로 끝까지 늘리는 버튼은 아닙니다.
사용자가 정한 목표 회차 안에서만 복구하거나 줄입니다.

예를 들어 10회차를 설정했는데 10회차 전체를 살 돈이 1 CR 부족하면, 억지로 10회차를 시작하지 않습니다.
가능한 회차까지만 줄입니다.
이때 스바루작이면 구매 33대, 강화 33대, 삭제 32대 같은 1회차 구성은 먼저 유지하고 회차를 줄입니다.

1회차 전체도 살 수 없지만 차량 몇 대는 살 수 있다면, 회차는 1회로 두고 구매 대수만 가능한 만큼 줄입니다.
강화 대수는 구매 대수를 넘지 않게 줄고, 삭제 대수는 `구매 대수 - 1`을 넘지 않게 줄어듭니다.

차량 1대도 살 수 없으면 실행하지 않습니다.

## 돈이 거의 없을 때의 루트

초반 루트는 이렇게 생각하면 됩니다.

```text
작업차 1대 준비
-> 의상/경적/제스처를 가능한 만큼 구매
-> 보유CR 입력
-> 적용
-> 아주 작은 규모로 실행
-> 얻은 휠스핀으로 CR 확보
-> 다시 보유CR 입력
-> 적용
-> 구매 대수 또는 회차를 조금 늘림
```

이때 중요한 건 욕심을 줄이는 것입니다.

처음부터 33대를 맞추려고 하지 않아도 됩니다.
2대, 3대, 5대부터 시작해도 됩니다.
작게 돌려서 휠스핀을 얻고, 그 돈으로 다음 실행을 키우면 됩니다.

이 방식은 느리지만 안전합니다.
돈이 없는 계정일수록 안전이 먼저입니다.

## 1사이클이 가능해지는 기준

스바루작 1사이클은 기본값 기준으로 구매 33대입니다.

```text
81,700 CR x 33대 x 1회 = 2,696,100 CR
```

즉, 스바루작을 기본값 그대로 1회차 돌리려면 차량 구매비만 약 `2,696,100 CR`이 필요합니다.
여기에 의상/경적/제스처를 사는 비용과 약간의 여유 CR이 있으면 더 좋습니다.

보유 CR이 이보다 적으면 실패가 아니라 작은 시작입니다.
`적용` 버튼을 누르고 가능한 구매 대수로 줄여서 시작하면 됩니다.

## 언제 올인원을 돌려도 되나?

아래 조건을 만족하면 올인원을 시도해볼 수 있습니다.

- 테스트 탭의 `최소런`이 정상 통과했습니다.
- 테스트 탭의 `권장런`도 정상 통과했습니다.
- 작업차 선택, 구매, 강화, 삭제 흐름이 내 PC에서 맞습니다.
- 보유 CR을 입력했고 `적용`으로 값이 확인됐습니다.
- 차량 1대도 못 사는 상태가 아닙니다.
- 실행 중 이상하면 `F2`로 멈출 준비가 되어 있습니다.

처음 올인원을 돌릴 때도 큰 회차를 추천하지 않습니다.
먼저 1회차 또는 작은 구매 대수로 확인하는 편이 좋습니다.

## 하지 말아야 할 것

아래는 초보자가 가장 쉽게 사고 내는 패턴입니다.

- 보유 CR을 입력하지 않고 올인원을 시작하기
- 1사이클 살 돈이 없는데 기본값 그대로 여러 회차를 돌리기
- 테스트런 없이 바로 긴 올인원을 돌리기
- 휠스핀을 직접 까거나 새 차를 얻은 뒤 최근 추가됨 목록을 확인하지 않기
- 작업 중 키보드, 마우스, 게임패드를 만지기
- F2를 눌러야 할 화면에서 "조금 더 보면 되겠지" 하고 기다리기

특히 차량 구매 중 돈이 떨어지는 상황은 위험합니다.
구매가 중간에 실패하면 이후 강화/삭제 대상이 예상과 달라질 수 있습니다.
그래서 `보유CR` 입력과 `적용` 버튼이 중요합니다.

## 아직 더 채워야 할 부분

이 문서는 시작 루트 초안입니다.
아직 아래 내용은 더 확인이 필요합니다.

- 의상, 경적, 제스처를 대략 얼마에 정리할 수 있는지
- 완전 무자본 계정이 첫 작업차 1대와 초기 CR을 모으는 가장 쉬운 방법
- 초보자 기준으로 추천할 첫 목표 CR
- 스바루작을 시작하기 전 최소 권장 스킬 포인트

확정되지 않은 숫자를 억지로 적으면 오히려 위험합니다.
이 문서는 먼저 "돈이 없으면 작게 시작한다"는 방향을 잡기 위한 초안입니다.
