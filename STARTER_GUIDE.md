# v0.5.0-rc.1 스바루런 시작 가이드

## 1. 새 폴더

이 버전은 새 챌린지 경로를 검증하는 공개 RC입니다. 기존 `v0.4.3` 폴더에 덮어쓰지 말고 별도 폴더에 압축을 풀어 시험합니다.

## 2. 버전 확인

프로그램 파일 버전이 `0.5.0-rc.1`인지 확인합니다. v0.4.3 실행 파일에는 새 챌린지 경로가 없습니다.

## 3. 기준 환경

- Windows x64
- FH6 한국어 UI
- FHD 1920 x 1080
- 전체 화면 또는 창 전체 화면
- 120 FPS 권장
- `기술`: 켜기 권장 — 10 포인트 적립 확인용
- `다음에 할 일`: 꺼짐
- `포커스 상실 시 일시 정지`: 끄기

게임 창은 계속 보이는 상태로 두고 실행 중 다른 창으로 전환하지 마세요.

## 4. 스바루런 준비

- EventLab > `챌린지 플레이하기`
- 공유 코드 `103 435 586`
- 필수 작업차 `1998 SUBARU IMPREZA 22B-STI VERSION`
- 마지막 회차의 두 번째 Enter가 필수 좋아요를 확정하는지 확인
- 1회 직접 완료해 `10 SP` 지급 확인

챌린지는 반드시 스바루 작업차로 진행합니다. 차량 OCR과 자동 재탑승은 사용하지 않으므로 사용자가 직접 `1998 SUBARU IMPREZA 22B-STI VERSION`에 탑승해야 합니다. 단독 `스바루런`은 작업차로 챌린지 1회차가 이미 시작된 주행 화면에서 F1을 누릅니다. 테스트·올인원은 작업차를 탄 오픈월드에서 시작합니다.

장애물에 강하게 충돌해 속도가 크게 떨어지면 맵의 실패 조건 때문에 챌린지가 실패할 수 있습니다. 이 경우 즉시 `F2`로 정지하고 정상 화면으로 수동 복구한 뒤 다시 시작합니다.

## 5. FH6 398.92.0 패치 경고

- 22B 1대: `313,500 CR` / `30 SP` / 슈퍼 휠스핀 1회
- 스바루 프리셋 33대: `10,345,500 CR` / `990 SP` / 슈퍼 휠스핀 33회
- 구 EventLab 이벤트: 사용자 실측상 `50,000`점 = 최대 `1 SP`, 신 버전에서 폐기
- 신 챌린지 코드 `103 435 586`: 현재 수동 검증 기준 1회 `10 SP`
- 기본 99회: `990 SP`, 메뉴 입력 제외 약 83분

기본 흐름은 `W`를 계속 누른 채 28초 주행 후 `Esc`로 재시작하고 22초 로딩을 기다립니다. 마지막 회차는 `결과 진행 Enter → 3초 → 좋아요 확정 Enter`로 끝냅니다. 후속 패치로 보상이 바뀔 수 있으므로 대량 실행 전에 반드시 테스트합니다.

## 6. 먼저 알아둘 키

- `F1`: 실행
- `F2`: 중지 요청
- `F3`: 중지 요청 후 종료
- `F4`: 휠스핀 진단 캡처

## 7. 첫 시험

1. 오픈월드에서 `테스트` 탭을 엽니다.
2. `챌린지만` 프리셋을 선택합니다. 기본값은 `런 1 / 구매 0 / 강화 0 / 삭제 0`입니다.
3. F1을 누르고 코드 검색, W 유지, 28초 주행, 마지막 Enter 2회, 오픈월드 복귀를 직접 봅니다.
4. 시작 전후 SP를 확인해 `10 SP` 지급을 검증합니다.
5. 이 구간이 성공한 뒤에만 `연결검증`으로 `런 1 / 구매 1 / 강화 1 / 삭제 0`을 시험합니다.
6. 삭제·자동 판매는 마지막에 각각 1회만 별도로 시험합니다.

이 프로그램의 이미지/OCR 판정은 제한적인 추정입니다. 원하는 차량의 구매·삭제·판매를 보증하지 않습니다.

## 8. 문제가 생기면

`F2`로 멈추고 `log/last-run-status.txt`와 가장 최근 `session-*.log`를 확인합니다. 로그나 PNG를 공유하기 전 Windows 사용자 이름, 로컬 경로, 게이머태그, CR 등 계정 정보를 지우거나 가리세요.

상세 설명은 `README.md`와 `GUIDE.md`에 있습니다.
