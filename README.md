# FH6 Subaruu Releases

FH6 스바루런, 차량 구매, 자동차 마스터리, 삭제 루프를 보조하는 비공식 Windows x64 배포 저장소입니다.
이 저장소는 배포용 ZIP, 릴리즈 노트, 사용 안내만 제공합니다.

소스 코드, 개발 로그, 개인 작업 내역은 포함하지 않습니다.
사용은 전적으로 사용자 본인 책임입니다.
특히 한국 시간 기준 매주 목요일 23:30 전후에는 새 시즌/차량 갱신으로 제조사 위치와 차량 위치가 달라질 수 있습니다.
이 시간 이후에는 기존 이동횟수를 그대로 믿지 말고, 틀린 차량을 구매하지 않도록 반드시 작은 테스트런부터 다시 확인하세요.

자세한 초보자 설명은 [GUIDE.md](GUIDE.md)를 먼저 읽어주세요.

## 최신 다운로드

최신 릴리즈:

- [Single-file guide/icon/test preset build 2026-06-29](https://github.com/unmatched785/fh6-subaruu-release/releases/tag/singlefile-guide-icon-testpreset-20260629)
- 파일: `subaruu.zip`
- SHA256: `B902F6B9383DF695C203E9F3F70B240735EF30646FEB79FCC84CB0C11A965847`

ZIP을 새 폴더에 풀고 `subaruu.exe`를 실행하세요.

## 반드시 먼저 읽을 것

- `F1`: 현재 선택된 탭 실행
- `F2`: 긴급 정지
- `F3`: 긴급 정지 후 프로그램 종료

`F1`은 확인창 없이 바로 실행합니다.
화면이 예상과 다르거나 자동차 매장/상품권/스토어/현금 결제 화면이 뜨면 즉시 `F2`를 누르세요.
더 위험하면 `F3`으로 프로그램을 종료하세요.

매주 목요일 23:30 전후에는 제조사/차량 위치가 바뀔 수 있습니다.
그 이후 첫 실행은 반드시 `최소런` 또는 개별 탭으로 다시 검증하세요.

처음 쓰는 환경에서는 바로 `올인원`을 돌리지 마세요.

```text
최소런 -> 권장런 -> 실패 지점 개별 탭 확인 -> 올인원
```

테스트런과 개별 탭은 장식이 아니라 안전장치입니다.

## 시작 전 준비

1. FH6를 실행하고 오픈월드 상태로 둡니다.
2. 스바루작 이벤트랩 맵을 즐겨찾기해 둡니다.
3. 작업차 `1998 SUBARU IMPREZA 22B-STI VERSION`을 즐겨찾기해 둡니다.
4. 작업차 즐겨찾기는 가능하면 해당 1대만 남깁니다.
5. 아래 필수 게임 설정을 적용합니다.
6. 구매 위치는 본인 차고와 DLC 보유 상태 기준으로 직접 확인합니다.
7. 작업 중에는 게임패드, 키보드, 마우스로 다른 입력을 넣지 않습니다.

스바루작 이벤트랩:

- 맵 코드를 직접 입력할 수 있으면 `113938786`을 사용합니다.
- Xbox/Game Pass에서 가상 키보드가 뜨면 이 맵만 따로 즐겨찾기하고 `맵 코드` 칸은 비워두는 방식을 권장합니다.

작업차 튜닝:

- `1998 SUBARU IMPREZA 22B-STI VERSION`
- 튜닝 코드: `146241846`
- 작업차 자동차 마스터리에서 아래 핵심 스킬 3개를 먼저 찍습니다.
  - `멀티 맥서`: 기술 멀티플라이어가 9까지 증가
  - `즐거움의 시작`: 기술 멀티플라이어가 2.5배 빨리 증가
  - `목숨 추가`: 2회 충돌해야 연쇄 기술이 깨짐. 콤보가 깨져 999포인트 누적에 실패하는 상황을 막기 위한 사전 방지용

필수 게임 설정:

| 메뉴 | 항목 | 값 |
| --- | --- | --- |
| 난이도 | 주행 보조 사전 설정 | 보조 최대 |
| HUD 및 게임플레이 | 기술 | 꺼짐 |
| 비디오/그래픽 | 해상도 | FHD, 1920 x 1080 |

이 설정을 적용해야 정상 동작을 전제로 볼 수 있습니다.
특히 해상도와 UI 표시가 다르면 이미지 인식 위치가 달라질 수 있습니다.

Xbox/Game Pass 환경에서 공유코드 입력창에 가상 키보드가 뜨면 `맵 코드` 칸을 비워두세요.
맵 코드가 비어 있으면 공유코드 입력창을 열지 않고 게임 안의 `내 즐겨찾기` 경로를 사용합니다.

## 정상 시작 위치

구매 탭:

```text
자동차 컬렉션 > 제조사 > 자동차 선택 화면
```

슈퍼휠/삭제 탭:

```text
페스티벌 > 경매장 > 경매 시작 > 차량 선택 목록
```

`자동차 매장`, `상품권`, `스토어`, `현금 결제` 화면은 정상 구매 경로가 아닙니다.
이 화면이 나오면 즉시 `F2`로 멈추세요.

## 추천 사용 흐름

1. `테스트` 탭에서 `최소런`을 실행합니다.
2. 이벤트랩 진입, 작업차 선택, 구매, 슈퍼휠 진입이 이어지는지 확인합니다.
3. `테스트` 탭에서 `권장런`을 실행합니다.
4. 구매 위치와 삭제 대상이 맞는지 확인합니다.
5. 실패한 지점이 있으면 `스바루런`, `구매`, `슈퍼휠`, `삭제` 개별 탭으로 그 구간만 다시 확인합니다.
6. 문제가 없을 때만 `올인원` 탭을 실행합니다.

테스트런 기준:

- `최소런`: 런 1회, 구매 2대, 강화 2대, 삭제 1대, 회차 1회
- `권장런`: 런 2회, 구매 3대, 강화 3대, 삭제 2대, 회차 2회

`최소런`은 4분짜리 100포인트 맵을 1회 돌았을 때의 기본 확인용입니다.
레부엘토작은 1대당 39포인트라서 100포인트로 최대 2대까지 강화할 수 있고, 삭제 1대까지 테스트할 수 있습니다.
`권장런`은 2회차 요소를 넣어서 다음 회차 재진입과 이어서 작동하는 흐름까지 확인합니다.

## 탭별 한 줄 설명

- `스바루런`: 이미 이벤트랩 주행 화면에 들어간 상태에서 주행만 반복합니다.
- `구매`: 자동차 컬렉션의 차량 카드 화면에서 설정된 횟수만큼 구매 입력을 반복합니다.
- `슈퍼휠`: 최근 추가됨 기준으로 차량을 탑승하고 자동차 마스터리를 찍습니다.
- `삭제`: 최근 추가됨 기준 1번 칸부터 차량을 삭제합니다.
- `테스트`: 올인원과 같은 경로를 짧은 횟수로 검증합니다.
- `올인원`: 오픈월드에서 시작해 이벤트랩, 주행, 구매, 강화, 삭제를 이어갑니다.

슈퍼휠 탭의 `스바루작`, `라부엘토작`, `마즈다작`은 프리셋 버튼입니다.
버튼을 누르면 구매/강화/삭제 횟수와 자동차 마스터리 경로가 자동으로 채워집니다.
프리셋 버튼만 누른다고 실행되는 것은 아니며, 실제 시작은 항상 `F1`입니다.

탭별 자세한 시작 화면과 복구법은 [GUIDE.md](GUIDE.md)를 보세요.

## 배포 파일

- `subaruu.exe`: 실행 파일
- `subaruu.default.conf`: 첫 실행용 기본 설정
- `eventlab-templates/`: 이미지 인식 템플릿
- `README.md`: 빠른 안내
- `GUIDE.md`: 초보자용 상세 가이드
- `SHA256SUMS.txt`: 패키지 내부 파일 해시

`subaruu.conf`는 실행 후 같은 폴더에 생성되는 사용자 설정 파일입니다.
배포 ZIP에는 `subaruu.conf`를 넣지 않습니다.
새 버전을 기존 폴더에 덮어 풀어도 저장해 둔 설정값을 덮어쓰지 않기 위한 처리입니다.
