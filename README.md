# 스바루런 보조 도구

FH6 스바루런, 차량 구매, 자동차 마스터리, 삭제 루프와 일반·슈퍼 휠스핀 처리를 보조하는 비공식 Windows x64 도구입니다.
키 입력과 화면 캡처 기반으로 동작합니다.

사용은 전적으로 사용자 본인 책임입니다.
게임 업데이트, 해상도, DLC 보유 상태, 차고 상태, 즐겨찾기 목록, PC 로딩 속도에 따라 메뉴 위치와 대기 시간이 달라질 수 있습니다.
특히 한국 시간 기준 매주 목요일 23:30 전후에는 새 시즌/차량 갱신으로 제조사 위치와 차량 위치가 달라질 수 있습니다.
이 시간 이후에는 기존 이동횟수를 그대로 믿지 말고, 틀린 차량을 구매하지 않도록 반드시 작은 테스트런부터 다시 확인하세요.

처음 사용하는 경우 이 README의 준비 사항과 최소런 순서를 먼저 확인하세요.

처음 받는 사람은 GitHub Releases의 `Latest` 릴리즈만 받으세요.
현재 권장판은 2026-07-13 휠스핀 보상 장부 릴리즈입니다.

휠스핀은 아직 테스트런·올인원에 연결되어 있지 않습니다.
Forza Edition 차량은 FE 카드 표식과 8종 후보 목록을 먼저 확인하며, 판정이 애매해도 판매하지 않고 보관합니다.
`상세 화면 저장`은 기본 OFF이며 정상 회차는 텍스트 로그만 남깁니다. 오류·안전정지·F4 수동 캡처는 항상 PNG로 저장합니다.

## 반드시 먼저 읽을 것

- `F1`: 현재 선택된 탭 실행
- `F2`: 긴급 정지
- `F3`: 긴급 정지 후 프로그램 종료

`F1`은 확인창 없이 바로 실행합니다.
화면이 예상과 다르거나 자동차 매장/상품권/스토어/현금 결제 화면이 뜨면 즉시 `F2`를 누르세요.
더 위험하면 `F3`으로 프로그램을 종료하세요.

매주 목요일 23:30 전후에는 제조사/차량 위치가 바뀔 수 있습니다.
그 이후 첫 실행은 반드시 `최소런` 또는 개별 탭으로 다시 검증하세요.

처음 쓰는 환경에서는 바로 `올인원`을 돌리지 마세요.

```text
최소런 -> 권장런 -> 실패 지점 개별 탭 확인 -> 올인원
```

테스트런과 개별 탭은 장식이 아니라 안전장치입니다.
본인 PC, 화면비, 메뉴 위치, 차량 보유 상태가 맞는지 먼저 확인하세요.

## 다운로드와 실행

배포 ZIP을 풀면 `subaruu` 폴더가 생깁니다.
그 안의 `subaruu.exe`를 실행합니다.
Windows x64 self-contained single-file 빌드라서 별도 .NET Desktop Runtime 설치는 필요하지 않습니다.
이미지 인식을 위해 `data`와 `x64` 폴더는 `subaruu.exe` 옆에 그대로 두세요.

`subaruu` 폴더 구성:

- `subaruu.exe`: 실행 파일
- `config/`: 기본 설정과 사용자 설정
- `data/eventlab-templates/`: 이벤트랩·차량 이미지 인식 자료
- `data/tessdata/`: 휠스핀 보상 OCR 자료
- `x64/`: OCR 실행용 파일
- `result/`: 휠스핀 전체 요약·상세 장부 RTF와 검색·복사용 TXT
- `README.md`: 빠른 안내
- `SHA256SUMS.txt`: 패키지 내부 파일 해시

`config/subaruu.conf`는 실행 후 생성되는 사용자 설정 파일입니다.
배포 ZIP에는 사용자용 `subaruu.conf`를 넣지 않습니다.
새 버전을 기존 폴더에 덮어 풀어도 저장해 둔 설정값을 덮어쓰지 않기 위한 처리입니다.
기존 안정판 폴더의 `subaruu.conf`가 있으면 첫 실행 때 `config/subaruu.conf`로 자동 복사합니다.

실행하면 같은 폴더에 `log` 폴더가 생기고 `session-날짜-시간.log` 사용 로그가 저장됩니다.
가장 최근 실행 상태는 `log/last-run-status.txt`에도 따로 저장됩니다.
문제가 생겼을 때는 `log` 폴더와 `config/subaruu.conf`를 함께 보내면 원인 확인이 훨씬 쉬워집니다.

첫 실행 기본값은 `스바루작` 기준입니다.
`마즈다작`은 카패스 DLC 차량을 쓰는 선택 프리셋이므로, 본인이 해당 차량을 쓸 수 있을 때만 마스터리 탭에서 직접 누르세요.

## 시작 전 준비

1. FH6를 실행하고 오픈월드 상태로 둡니다.
2. 스바루작 이벤트랩 맵을 즐겨찾기해 둡니다.
3. 작업차 `1998 SUBARU IMPREZA 22B-STI VERSION`을 즐겨찾기해 둡니다.
4. 작업차 즐겨찾기는 가능하면 해당 1대만 남깁니다.
5. 아래 필수 게임 설정을 적용합니다.
6. 구매 위치는 본인 차고와 DLC 보유 상태 기준으로 직접 확인합니다.
7. 작업 중에는 게임패드, 키보드, 마우스로 다른 입력을 넣지 않습니다.

스바루작 이벤트랩:

- 맵 코드를 직접 입력할 수 있으면 `113938786`을 사용합니다.
- Xbox/Game Pass에서 가상 키보드가 뜨면 이 맵만 따로 즐겨찾기하고 `맵 코드` 칸은 비워두는 방식을 권장합니다.

작업차 튜닝:

- `1998 SUBARU IMPREZA 22B-STI VERSION`
- 튜닝 코드: `171 460 155`
- 작업차 자동차 마스터리에서 아래 핵심 스킬 6개를 먼저 찍습니다.
  - `멀티 맥서`: 기술 멀티플라이어가 9까지 증가
  - `즐거움의 시작`: 기술 멀티플라이어가 2.5배 빨리 증가
  - `실력을 보여줘`: 콤보 기술 점수 20% 증가
  - `밥상 엎기`: 기물파손 기술 점수 10% 증가
  - `격파 뺑소니`: 길거리 청소 기술 점수 15% 증가
  - `목숨 추가`: 2회 충돌해야 연쇄 기술이 깨짐. 콤보가 깨져 999포인트 누적에 실패하는 상황을 막기 위한 사전 방지용

필수 게임 설정:

| 메뉴 | 항목 | 값 |
| --- | --- | --- |
| 난이도 | 주행 보조 사전 설정 | 보조 최대 |
| HUD 및 게임플레이 | 기술 | 꺼짐 |
| HUD 및 게임플레이 | 다음에 할 일 | 꺼짐 |
| 비디오/그래픽 | 화면 모드 | 전체 화면 또는 창 전체 화면 |
| 비디오/그래픽 | 해상도 | FHD, 1920 x 1080 |
| 비디오/그래픽 | 프레임 제한 | 120 FPS 권장 |

이 설정을 적용해야 정상 동작을 전제로 볼 수 있습니다.
일반 창 모드는 아직 테스트하지 않았으므로 정상 동작을 보장하지 않습니다.
특히 해상도와 UI 표시가 다르면 이미지 인식 위치가 달라질 수 있습니다.
프레임도 이벤트랩 주행 속도에 영향을 줄 수 있습니다.
위 작업차 튜닝 기준 FHD 정상 주행 시간은 한 판 약 4분 1초대입니다.
프로그램 기본 주행 시간은 245초입니다.
올인원 보정 주행은 120초로, 약 50포인트를 추가 확보하는 안전마진입니다.
한 판이 5~6분대로 길어지면 프레임을 적절히 맞춘 뒤 다시 테스트하세요.
`다음에 할 일`이 켜져 있으면 오픈월드 복귀 후 이벤트 추천 카드가 떠서 구매 메뉴 진입을 막을 수 있습니다.

Xbox/Game Pass 환경에서 공유코드 입력창에 가상 키보드가 뜨면 `맵 코드` 칸을 비워두세요.
맵 코드가 비어 있으면 공유코드 입력창을 열지 않고 게임 안의 `내 즐겨찾기` 경로를 사용합니다.

## 정상 시작 위치

구매 탭:

```text
자동차 컬렉션 > 제조사 > 자동차 선택 화면
```

마스터리/삭제 탭:

```text
페스티벌 > 경매장 > 경매 시작 > 차량 선택 목록
```

테스트런/올인원에서 구매 후 `경매 시작` 대신 바로 윗칸으로 들어가면 `마스터리` 탭의 `경매장 대기`를 8~12초로 늘려 다시 확인하세요.
구매 후 차량 선택 목록을 확인하지 못하면 프로그램이 멈추는 것이 정상입니다.

`자동차 매장`, `상품권`, `스토어`, `현금 결제` 화면은 정상 구매 경로가 아닙니다.
이 화면이 나오면 즉시 `F2`로 멈추세요.

## 추천 사용 흐름

1. `테스트` 탭에서 `최소런`을 실행합니다.
2. 이벤트랩 진입, 작업차 선택, 구매, 마스터리 진입이 이어지는지 확인합니다.
3. `테스트` 탭에서 `권장런`을 실행합니다.
4. 구매 위치와 삭제 대상이 맞는지 확인합니다.
5. 실패한 지점이 있으면 `스바루런`, `구매`, `마스터리`, `삭제` 개별 탭으로 그 구간만 다시 확인합니다.
6. 문제가 없을 때만 `올인원` 탭을 실행합니다.

테스트 탭의 `맵` 칸은 테스트런 전용 맵코드입니다.
비워두면 공유코드 입력창을 열지 않고 게임 안의 `내 즐겨찾기` 경로를 사용합니다.

테스트런 기준:

- `최소런`: 런 1회, 구매 2대, 강화 2대, 삭제 1대, 회차 1회
- `권장런`: 런 2회, 구매 3대, 강화 3대, 삭제 2대, 회차 2회

`최소런`은 4분짜리 100포인트 맵을 1회 돌았을 때의 기본 확인용입니다.
레부엘토작은 1대당 39포인트라서 100포인트로 최대 2대까지 강화할 수 있고, 삭제 1대까지 테스트할 수 있습니다.
`권장런`은 2회차 요소를 넣어서 다음 회차 재진입과 이어서 작동하는 흐름까지 확인합니다.

중간에 슈퍼휠스핀을 직접 까거나 다른 차량을 새로 얻으면 `최근 추가됨` 정렬과 삭제 대상이 꼬일 수 있습니다.

## 탭별 한 줄 설명

- `스바루런`: 이미 이벤트랩 주행 화면에 들어간 상태에서 주행만 반복합니다. 주행 중에는 `W` 키만 누릅니다.
- `구매`: 자동차 컬렉션의 차량 카드 화면에서 설정된 횟수만큼 구매 입력을 반복합니다.
- `마스터리`: 최근 추가됨 기준으로 차량을 탑승하고 자동차 마스터리를 찍습니다.
- `삭제`: 최근 추가됨 기준 1번 칸부터 차량을 삭제합니다.
- `휠스핀`: 일반 또는 슈퍼 휠스핀을 독립 실행하고 중복 차량의 보관·판매를 판정합니다.
- `테스트`: 올인원과 같은 경로를 짧은 횟수로 검증합니다.
- `올인원`: 오픈월드에서 시작해 이벤트랩, 주행, 구매, 강화, 삭제를 이어갑니다.

현재 `휠스핀` 탭은 독립 실행 전용이며 테스트런·올인원에는 연결되어 있지 않습니다.

삭제 개별탭은 7월 4일 기준 입력 흐름을 그대로 사용합니다. 테스트와 올인원에서만 삭제 직전에 현재 프리셋의 연식·제조사(`1998 SUBARU`, `2024 LAMBORGHINI`, `1974 MAZDA`)를 확인하며, 불일치하면 해당 차량에 삭제 입력을 보내기 전에 멈춥니다. 차량 외형과 `신규` 배지는 차단 조건으로 사용하지 않습니다.

마스터리 탭의 `스바루작`, `라부엘토작`, `마즈다작`은 프리셋 버튼입니다.
버튼을 누르면 구매/강화/삭제 횟수, 자동차 마스터리 경로, 제조사/구매차 이동횟수가 자동으로 채워집니다.
프리셋 버튼만 누른다고 실행되는 것은 아니며, 실제 시작은 항상 `F1`입니다.
배포 기본값은 `스바루작`이며, `마즈다작`은 카패스 DLC 보유자용 선택 프리셋입니다.

구매 탭의 `진입`과 `간격`은 테스트런/올인원에서 오픈월드 메뉴를 열고 자동차 컬렉션으로 들어갈 때도 사용됩니다.
`진입`은 `Esc` 후 메뉴가 뜰 때까지 기다리는 시간이고, `간격`은 방향키/Enter 반복 입력 사이의 최소 간격입니다.

아래 시작 화면과 복구법을 확인한 뒤 작은 횟수부터 실행하세요.

## 휠스핀 탭

`휠스핀` 탭은 일반 또는 슈퍼 휠스핀을 설정한 횟수만큼 독립 처리합니다.

시작 위치:

```text
나의 HORIZON > 실행할 일반/슈퍼 휠스핀 카드에 포커스
```

- `레전더리 보관`: 기본 켜짐
- `에픽 보관`: 기본 꺼짐
- `의상,경적 올클`: 의상과 경적을 모두 획득한 계정만 사용
- 독점 차량: 등급과 무관하게 보관
- 비독점 중복 차량: 보관 옵션에 해당하지 않으면 CR로 판매
- CR·의상·경적: 판매 대상이 아니며 그대로 수령

목표 횟수를 완료하면 휠 메뉴에 머무릅니다. 오픈월드로 자동 복귀하지 않습니다.
상태 줄에서 슬롯 색상, `CR/차량/비차량`, 독점 여부와 최종 행동을 확인할 수 있습니다.
완료 후 `전체 요약`에는 통계와 주요 보상을, `상세 로그`에는 휠스핀 1회당 한 줄을 표시합니다. 슈퍼 휠스핀의 보상 3개도 같은 회차 줄에 묶습니다.
보상 기록은 실제 `subaruu.exe` 옆 `result/wheelspin-날짜-시간-종류/`에 저장되며 내부 임시 추출 폴더는 사용하지 않습니다.
현재 휠스핀 처리는 테스트런·올인원에 연결되어 있지 않습니다.

## 자주 터지는 지점

### 문제를 보고해야 함

`subaruu.exe` 옆의 `log` 폴더와 `config/subaruu.conf`를 같이 보내주세요.
사용 로그에는 실행한 탭, 주요 설정값, 진행 상태, 중단 이유, 이벤트랩 인식 실패 캡처 경로가 남습니다.
`last-run-status.txt`에는 마지막 실행의 탭, 회차, 진행 단계, 마지막 정상 지점, 정지 사유가 요약됩니다.

### 가상 키보드가 떠서 맵 코드 입력이 안 됨

맵 코드를 비우고 이벤트랩을 즐겨찾기에 넣어 사용하세요.

### 제조사 검색에서 멈춘 것처럼 보임

느린 PC에서 화면 전환 전에 입력이 들어간 경우일 수 있습니다.
작업차 선택 후 이벤트맵 로딩에서 멈추거나, 스바루런 후 오픈월드로 나왔는데 구매 메뉴로 못 들어가면 `스바루런` 탭의 `로딩 대기`를 30~45초로 늘려보세요.
이 값은 이벤트랩 로딩, 스바루런 후 구매 위치 진입 전 대기, 보정런 후 오픈월드 복귀 대기에 쓰입니다.
기본값은 25초입니다.
빠른 PC에서는 테스트런으로 확인한 뒤 20초까지 줄여도 되고, 느린 PC에서는 30~45초로 늘려보세요.

### 스바루런 후 이벤트 카드 화면이 뜸

게임 설정에서 `HUD 및 게임플레이 > 다음에 할 일`이 `꺼짐`인지 먼저 확인하세요.
이 설정이 켜져 있으면 오픈월드 복귀 후 이벤트 추천 카드가 뜰 수 있습니다.
오픈월드 메뉴에서 구매 메뉴로 이동하는 첫 방향키 입력이 씹힌 경우일 수 있습니다.
구매 탭의 `간격`을 먼저 0.5~0.7초로 올려 테스트하세요.
그래도 반복되면 `진입`을 8~10초로 늘리고, `log` 폴더에서 `purchase-menu-open-wait` 기록을 확인하세요.

### 경매 시작 대신 윗칸으로 들어감

구매 후 경매장 화면 전환이 느리면 `경매 시작`으로 내려가는 입력이 씹혀 윗칸이 선택될 수 있습니다.
`마스터리` 탭의 `경매장 대기` 기본값은 8초입니다.
그래도 반복되면 12초로 늘려 테스트하세요.
차량 선택 목록을 확인하지 못하면 마스터리/삭제로 넘어가지 않고 멈추며, 실패 캡처와 로그가 `log` 폴더에 남습니다.
구매 후 경매장으로 들어가야 하는데 자동차 매장으로 빠지는 문제도 `경매장 대기`를 늘려야 합니다.

### 구매 차량이 다름

제조사/구매차 이동횟수가 본인 화면과 맞지 않는 상태입니다.
구매 위치는 반드시 직접 확인해야 합니다.

### 삭제 대상이 꼬임

작업 중 슈퍼휠스핀을 직접 까거나 다른 신규 차량을 받지 마세요.

### 이벤트랩 한 판이 5~6분대로 느림

프레임 제한이 주행 속도에 영향을 줄 수 있습니다.
위 작업차 튜닝 기준 FHD 스바루런 정상 시간은 한 판 약 4분 1초대입니다.
6분 안팎으로 나오면 먼저 비디오/그래픽 설정에서 프레임 제한을 120 FPS로 맞추고 다시 테스트하세요.
프레임을 맞춘 뒤에도 느리면 작업차 튜닝 코드 `171 460 155`와 작업차 핵심 마스터리 6개가 적용됐는지 확인하세요.

## 배포자 메모

이 도구는 비공식 개인 편의 도구입니다.
문제가 생기면 `F2`로 즉시 멈추고 작은 테스트 값으로 다시 검증하세요.
