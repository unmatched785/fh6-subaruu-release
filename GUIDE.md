# 스바루런 초보자 가이드

이 문서는 처음 쓰는 사람이 틀린 화면에서 `F1`을 누르는 사고를 줄이기 위한 상세 가이드입니다.
README는 빠른 체크리스트이고, 이 파일은 실제 사용 순서를 천천히 설명합니다.

한국 시간 기준 매주 목요일 23:30 전후에는 새 시즌/차량 갱신으로 제조사 위치와 차량 위치가 달라질 수 있습니다.
그 이후에는 지난주에 맞았던 이동횟수라도 틀린 차량을 구매할 수 있으니, 반드시 `최소런` 또는 개별 탭으로 다시 확인하세요.

## 1. 이 프로그램이 하는 일

이 도구는 FH6에서 반복 입력을 대신 눌러주는 비공식 보조 도구입니다.
게임 화면을 캡처해서 일부 메뉴와 차량 카드를 찾고, 키보드 입력을 보내서 이동합니다.

할 수 있는 일:

- 이벤트랩 스바루런 반복
- 자동차 컬렉션 안에서 차량 구매 입력 반복
- 최근 추가됨 기준 차량 탑승
- 자동차 마스터리 경로 입력
- 최근 추가됨 기준 차량 삭제
- 테스트런으로 짧은 검증
- 올인원으로 전체 흐름 반복

하지 않는 일:

- 모든 사용자 환경에서 성공 보장
- 바뀐 차량 위치 자동 추론
- 잘못 산 차량이나 삭제한 차량 복구
- 게임 업데이트 이후 메뉴 변화 자동 대응
- 자동차 매장/상품권/스토어 구매 보조

## 2. 제일 중요한 안전키

- `F1`: 현재 선택된 탭 실행
- `F2`: 긴급 정지
- `F3`: 긴급 정지 후 프로그램 종료

`F1`은 확인창 없이 바로 실행됩니다.
무언가 이상하면 고민하지 말고 `F2`를 누르세요.
상황이 더 꼬였거나 프로그램을 완전히 끄고 싶으면 `F3`을 누르세요.

다음 화면이 뜨면 정상 경로가 아닙니다.

- 자동차 매장
- 자동차 구입
- 상품권
- 스토어
- 현금 결제
- 게임 UI 입력창이 엉뚱하게 포커스를 가져간 화면

이런 화면이 보이면 즉시 `F2`입니다.

## 3. 처음 쓰는 순서

처음부터 긴 `올인원`을 돌리지 마세요.
안전한 순서는 아래입니다.

```text
최소런 -> 권장런 -> 실패 지점 개별 탭 확인 -> 올인원
```

현재 테스트런 기준은 아래처럼 잡습니다.

| 버튼 | 런 | 구매 | 강화 | 삭제 | 회차 | 목적 |
| --- | ---: | ---: | ---: | ---: | ---: | --- |
| 최소런 | 1 | 2 | 2 | 1 | 1 | 4분짜리 100포인트 맵 1회로 구매/강화/삭제 기본 흐름 확인 |
| 권장런 | 2 | 3 | 3 | 2 | 2 | 2회차 재진입과 이어서 작동하는지 확인 |

레부엘토작은 1대당 39포인트를 사용합니다.
100포인트 맵을 1회 돌면 최대 2대까지 강화할 수 있으므로, `최소런`은 구매 2대, 강화 2대, 삭제 1대로 둡니다.
`권장런`은 2회차 요소를 추가해서 다음 사이클 진입까지 보는 기준입니다.

매주 목요일 23:30 전후 또는 새 차량이 추가된 직후에는 이 순서를 다시 지켜야 합니다.
제조사/차량 위치가 바뀌면 기존 `제조사 이동횟수`와 `구매차 이동횟수`가 틀린 차량으로 향할 수 있습니다.

## 4. 시작 전 준비

게임 쪽 준비:

- FH6를 실행하고 오픈월드 상태로 둡니다.
- 스바루작 이벤트랩 맵을 즐겨찾기해 둡니다.
- 작업차 `1998 SUBARU IMPREZA 22B-STI VERSION`을 즐겨찾기해 둡니다.
- 가능하면 작업차 즐겨찾기는 해당 1대만 남깁니다.
- 필수 게임 설정을 적용합니다.
- 구매 위치는 본인 화면에서 직접 확인합니다.

스바루작 이벤트랩:

- 맵 코드를 직접 입력할 수 있으면 `113938786`을 사용합니다.
- Xbox/Game Pass에서 가상 키보드가 뜨면 이 맵만 따로 즐겨찾기하고 `맵 코드` 칸은 비워두는 방식을 권장합니다.
- 즐겨찾기 목록에 여러 이벤트랩이 섞여 있으면 다른 맵으로 들어갈 수 있으니, 가능하면 스바루작 맵만 따로 즐겨찾기하세요.

작업차 튜닝:

- 대상 차량: `1998 SUBARU IMPREZA 22B-STI VERSION`
- 튜닝 코드: `146241846`
- 작업차 튜닝이 다르면 주행 결과와 스킬포인트 수급이 달라질 수 있습니다.

필수 게임 설정:

| 메뉴 | 항목 | 값 |
| --- | --- | --- |
| 난이도 | 주행 보조 사전 설정 | 보조 최대 |
| HUD 및 게임플레이 | 기술 | 꺼짐 |
| 비디오/그래픽 | 해상도 | FHD, 1920 x 1080 |

이 세 가지는 권장이 아니라 정상 동작 전제입니다.
주행 보조가 다르면 이벤트랩 주행이 흔들릴 수 있고, `기술` 표시가 켜져 있으면 화면 위에 뜨는 알림이 인식과 입력 타이밍을 방해할 수 있습니다.
해상도가 FHD가 아니면 이미지 인식 기준 위치가 달라져 제조사/작업차 선택이 실패할 수 있습니다.

프로그램 쪽 준비:

- `맵 코드`를 비우면 내 즐겨찾기 경로를 사용합니다.
- Xbox/Game Pass에서 가상 키보드가 뜨면 맵 코드를 비우는 방식을 권장합니다.
- 구매 횟수, 강화 횟수, 삭제 횟수를 길게 잡기 전에 작은 값으로 테스트합니다.
- 실행 중에는 게임패드, 키보드, 마우스로 다른 입력을 넣지 않습니다.

## 5. 절대 시작하면 안 되는 화면

정상 구매 경로는 자동차 매장이 아닙니다.
정상 경로는 자동차 컬렉션입니다.

잘못된 화면:

- 자동차 매장 화면
- 자동차 구입 화면
- 상품권 구매 화면
- 스토어 연결 중 화면
- 현금 결제 관련 화면

이 화면에서 계속 진행하면 원하지 않는 구매 화면으로 들어갈 수 있습니다.
바로 `F2`를 누르고, 자동차 컬렉션으로 돌아가세요.

## 6. 탭별 시작 위치

### 스바루런

정상 시작 위치:

```text
이벤트랩 주행 화면
```

이미 이벤트랩 안에서 차량을 타고 주행을 시작할 수 있는 상태여야 합니다.

### 구매

정상 시작 위치:

```text
자동차 컬렉션 > 제조사 > 자동차 선택 화면
```

구매할 차량 카드에 커서가 올라가 있어야 합니다.
자동차 매장이나 상품권 화면이면 잘못된 경로입니다.

### 슈퍼휠

정상 시작 위치:

```text
페스티벌 > 경매장 > 경매 시작 > 차량 선택 목록
```

`경매 시작` 타일 위가 아닙니다.
차량 카드들이 보이는 목록 화면이어야 합니다.

슈퍼휠 탭의 `스바루작`, `라부엘토작`, `마즈다작`은 프리셋 버튼입니다.
직접 실행 버튼이 아니라, 아래 값을 한 번에 채우는 버튼입니다.

| 버튼 | 구매 | 강화 | 삭제상한 | 용도 |
| --- | ---: | ---: | ---: | --- |
| 스바루작 | 33 | 33 | 32 | 기존 SUBARU 작업 기준 |
| 라부엘토작 | 25 | 25 | 24 | 레부엘토 작업 기준 |
| 마즈다작 | 47 | 47 | 46 | 999포인트 기준 마즈다 작업 |

프리셋 버튼을 누른 뒤에도 바로 매크로가 실행되지는 않습니다.
값이 맞는지 확인하고, 정상 시작 위치인 `페스티벌 > 경매장 > 경매 시작 > 차량 선택 목록`에서 `F1`을 눌러야 실행됩니다.

### 삭제

정상 시작 위치:

```text
페스티벌 > 경매장 > 경매 시작 > 차량 선택 목록
```

삭제 탭은 최근 추가됨 기준 1번 칸을 삭제 대상으로 삼습니다.
반드시 1번 칸이 삭제할 차량인지 확인하세요.

### 테스트와 올인원

정상 시작 위치:

```text
오픈월드
```

테스트 탭은 올인원과 같은 경로를 짧은 횟수로 검증합니다.
올인원은 가장 길고 위험한 탭이므로 `최소런`과 `권장런` 통과 후 사용하세요.

## 7. 구매 이동횟수 찾는 법

구매 위치는 사람마다 다를 수 있습니다.
DLC 보유 상태, 차고 상태, 신규 차량 추가 여부에 따라 같은 차량도 위치가 달라질 수 있습니다.

확인 순서:

1. 게임에서 자동차 컬렉션으로 갑니다.
2. 제조사 선택 화면을 엽니다.
3. 시작 커서 위치에서 목표 제조사까지 몇 번 눌러야 하는지 셉니다.
4. 그 값을 `제조사 이동횟수`에 넣습니다.
5. 목표 제조사에 들어갑니다.
6. 목표 차량까지 몇 번 눌러야 하는지 셉니다.
7. 그 값을 `구매차 이동횟수`에 넣습니다.

공유된 값은 참고용입니다.
신규 차량이 들어오면 위치가 바뀔 수 있습니다.

## 8. 최근 추가됨 기준

차량을 여러 대 구매하면 방금 산 차량들이 최근 추가됨 쪽에 모입니다.
프로그램은 이 정렬을 기준으로 새로 산 차량들을 찾아갑니다.

작업 중에는 아래 행동을 피하세요.

- 슈퍼휠스핀을 직접 까기
- 다른 신규 차량 받기
- 다른 차량 구매하기
- 차고 정렬을 수동으로 흔들기

중간에 새 차량이 섞이면 강화 대상이나 삭제 대상이 꼬입니다.

## 9. Xbox/Game Pass 사용자

Xbox/Game Pass 환경에서는 공유코드 입력창에서 가상 키보드가 뜨는 경우가 있습니다.
이때 키 입력 포커스가 가상 키보드로 넘어가면 코드 입력이 꼬일 수 있습니다.

권장 방식:

1. 스바루작 이벤트랩 맵 `113938786`을 게임 안에서 즐겨찾기합니다.
2. 프로그램의 `맵 코드` 칸을 비웁니다.
3. 테스트런으로 내 즐겨찾기 경로가 들어가는지 확인합니다.

맵 코드가 비어 있을 때 프로그램은 아래 흐름을 사용합니다.

```text
이벤트랩 검색/찾기 화면
PageDown 7회
내 즐겨찾기 선택
Enter
3초 대기
Enter
3초 대기
차량 선택 화면
```

즐겨찾기에 스바루작 이벤트랩 맵이 없으면 이 경로는 실패합니다.
여러 이벤트랩을 즐겨찾기해 둔 상태라면 순서가 꼬일 수 있으니, 스바루작 맵만 따로 즐겨찾기하는 것을 권장합니다.

## 10. 자주 터지는 문제와 복구법

### 자동차 매장/상품권/스토어 화면이 뜸

잘못된 경로입니다.
즉시 `F2`로 멈추세요.
정상 구매 시작 위치는 자동차 컬렉션의 차량 선택 화면입니다.

### 가상 키보드가 떠서 맵 코드 입력이 안 됨

맵 코드를 비우고 이벤트랩 즐겨찾기 방식을 사용하세요.

### 제조사 검색에서 멈춘 것처럼 보임

게임 로딩이 느리거나 화면 전환 전에 입력이 들어간 경우일 수 있습니다.
작업차 선택 후 이벤트맵 로딩에서 멈춘다면 `로딩 대기`를 30~45초로 늘려보세요.

### 스킬포인트가 부족함

가장 먼저 작업차를 제대로 탔는지 확인하세요.
작업차가 아니면 포인트 수급이 크게 줄어듭니다.

### 구매 차량이 다름

제조사/구매차 이동횟수가 본인 화면과 맞지 않는 상태입니다.
본인 화면에서 제조사와 차량 위치를 다시 세세요.

### 삭제 대상이 꼬임

구매/강화 중간에 다른 새 차량이 섞였을 가능성이 큽니다.
작업 중 슈퍼휠스핀을 직접 까거나 다른 신규 차량을 받지 마세요.

- 지금 게임 화면이 맞는가?
- 구매/강화/삭제 횟수가 작은 테스트로 검증됐는가?

이 셋이 확실하지 않으면 올인원을 돌리지 마세요.
