# 스바루런 초보자 가이드

이 문서는 처음 쓰는 사람이 틀린 화면에서 `F1`을 누르는 사고를 줄이기 위한 상세 가이드입니다.
README는 빠른 체크리스트이고, 이 파일은 실제 사용 순서를 천천히 설명합니다.

처음 받는 사람은 GitHub Releases의 `Latest` 릴리즈만 받으세요.
이전 2026-06-29 릴리즈들은 빠른 핫픽스 기록이며, 최신 릴리즈가 앞선 수정들을 포함합니다.

한국 시간 기준 매주 목요일 23:30 전후에는 새 시즌/차량 갱신으로 제조사 위치와 차량 위치가 달라질 수 있습니다.
그 이후에는 지난주에 맞았던 이동횟수라도 틀린 차량을 구매할 수 있으니, 반드시 `최소런` 또는 개별 탭으로 다시 확인하세요.

## 1. 이 프로그램이 하는 일

이 도구는 FH6에서 반복 입력을 대신 눌러주는 비공식 보조 도구입니다.
게임 화면을 캡처해서 일부 메뉴와 차량 카드를 찾고, 키보드 입력을 보내서 이동합니다.

할 수 있는 일:

- 이벤트랩 스바루런 반복
- 자동차 컬렉션 안에서 차량 구매 입력 반복
- 최근 추가됨 기준 차량 탑승
- 자동차 마스터리 경로 입력
- 최근 추가됨 기준 차량 삭제
- 테스트런으로 짧은 검증
- 올인원으로 전체 흐름 반복

하지 않는 일:

- 모든 사용자 환경에서 성공 보장
- 바뀐 차량 위치 자동 추론
- 잘못 산 차량이나 삭제한 차량 복구
- 게임 업데이트 이후 메뉴 변화 자동 대응
- 자동차 매장/상품권/스토어 구매 보조

## 2. 제일 중요한 안전키

- `F1`: 현재 선택된 탭 실행
- `F2`: 긴급 정지
- `F3`: 긴급 정지 후 프로그램 종료

`F1`은 확인창 없이 바로 실행됩니다.
무언가 이상하면 고민하지 말고 `F2`를 누르세요.
상황이 더 꼬였거나 프로그램을 완전히 끄고 싶으면 `F3`을 누르세요.

다음 화면이 뜨면 정상 경로가 아닙니다.

- 자동차 매장
- 상품권
- 스토어
- 현금 결제
- 게임 UI 입력창이 엉뚱하게 포커스를 가져간 화면

이런 화면이 보이면 즉시 `F2`입니다.
정상 구매 경로는 `자동차 컬렉션 > 제조사 > 자동차 선택 화면`입니다.

## 3. 처음 쓰는 순서

처음부터 긴 `올인원`을 돌리지 마세요.
안전한 순서는 아래입니다.

```text
최소런 -> 권장런 -> 실패 지점 개별 탭 확인 -> 올인원
```

왜 이렇게 하냐면, 이 프로그램은 사용자의 화면 상태에 매우 민감하기 때문입니다.
해상도, DLC, 차고 상태, 즐겨찾기 목록, 로딩 속도 중 하나만 달라도 이동 횟수가 달라질 수 있습니다.

처음에는 이렇게 확인하세요.

1. `테스트` 탭에서 `최소런`을 누릅니다.
2. 이벤트랩 진입과 작업차 선택이 되는지 봅니다.
3. 구매 화면으로 가는지 봅니다.
4. 슈퍼휠/강화 화면으로 이어지는지 봅니다.
5. 문제가 있으면 `F2`로 멈춥니다.
6. 실패한 구간의 개별 탭으로 작은 횟수만 다시 확인합니다.
7. `권장런`까지 통과하면 그때 `올인원`을 씁니다.

테스트런과 개별 탭은 귀찮은 절차가 아니라 안전장치입니다.

현재 테스트런 기준은 아래처럼 잡습니다.

| 버튼 | 런 | 구매 | 강화 | 삭제 | 회차 | 목적 |
| --- | ---: | ---: | ---: | ---: | ---: | --- |
| 최소런 | 1 | 2 | 2 | 1 | 1 | 4분짜리 100포인트 맵 1회로 구매/강화/삭제 기본 흐름 확인 |
| 권장런 | 2 | 3 | 3 | 2 | 2 | 2회차 재진입과 이어서 작동하는지 확인 |

레부엘토작은 1대당 39포인트를 사용합니다.
100포인트 맵을 1회 돌면 최대 2대까지 강화할 수 있으므로, `최소런`은 구매 2대, 강화 2대, 삭제 1대로 둡니다.
`권장런`은 2회차 요소를 추가해서 다음 사이클 진입까지 보는 기준입니다.

매주 목요일 23:30 전후 또는 새 차량이 추가된 직후에는 이 순서를 다시 지켜야 합니다.
제조사/차량 위치가 바뀌면 기존 `제조사 이동횟수`와 `구매차 이동횟수`가 틀린 차량으로 향할 수 있습니다.

## 4. 시작 전 준비

게임 쪽 준비:

- FH6를 실행하고 오픈월드 상태로 둡니다.
- 스바루작 이벤트랩 맵을 즐겨찾기해 둡니다.
- 작업차 `1998 SUBARU IMPREZA 22B-STI VERSION`을 즐겨찾기해 둡니다.
- 가능하면 작업차 즐겨찾기는 해당 1대만 남깁니다.
- 필수 게임 설정을 적용합니다.
- 구매 위치는 본인 화면에서 직접 확인합니다.

스바루작 이벤트랩:

- 맵 코드를 직접 입력할 수 있으면 `113938786`을 사용합니다.
- Xbox/Game Pass에서 가상 키보드가 뜨면 이 맵만 따로 즐겨찾기하고 `맵 코드` 칸은 비워두는 방식을 권장합니다.
- 즐겨찾기 목록에 여러 이벤트랩이 섞여 있으면 다른 맵으로 들어갈 수 있으니, 가능하면 스바루작 맵만 따로 즐겨찾기하세요.

작업차 튜닝:

- 대상 차량: `1998 SUBARU IMPREZA 22B-STI VERSION`
- 튜닝 코드: `146241846`
- 작업차 튜닝이 다르면 주행 결과와 스킬포인트 수급이 달라질 수 있습니다.

작업차 핵심 자동차 마스터리:

| 스킬 | 효과 | 중요한 이유 |
| --- | --- | --- |
| 멀티 맥서 | 기술 멀티플라이어가 9까지 증가 | 같은 주행에서 더 많은 스킬 점수를 받기 위한 핵심입니다. |
| 즐거움의 시작 | 기술 멀티플라이어가 2.5배 빨리 증가 | 짧은 이벤트랩 주행 안에서 멀티플라이어를 빨리 올립니다. |
| 실력을 보여줘 | 콤보 기술 점수 20% 증가 | 이벤트랩 주행 중 쌓이는 콤보 기술 점수 자체를 올립니다. |
| 밥상 엎기 | 기물파손 기술 점수 10% 증가 | 코스에서 기물을 부수며 얻는 점수를 보강합니다. |
| 격파 뺑소니 | 길거리 청소 기술 점수 15% 증가 | 길거리 청소 판정으로 들어오는 점수를 보강합니다. |
| 목숨 추가 | 2회 충돌해야 연쇄 기술이 깨짐 | 만에 하나 콤보가 깨져 999포인트 누적에 실패하는 상황을 막기 위한 사전 방지용입니다. |

이 여섯 개는 스바루런 작업차에 미리 찍어두는 준비용 스킬입니다.
슈퍼휠 탭에서 구매 차량에 찍는 자동차 마스터리 프리셋과는 다른 개념입니다.
찍는 순서는 자동차 마스터리 화면의 연결선을 따라가면 됩니다.
중요한 기준은 최종적으로 위 여섯 스킬이 `보유 중` 상태인지 확인하는 것입니다.
작업차 마스터리가 빠져 있으면 같은 맵을 돌아도 스킬포인트 수급이 크게 줄거나, 충돌 한 번에 연쇄가 끊길 수 있습니다.
특히 `목숨 추가`가 없으면 작은 사고 한 번으로 콤보가 깨져 장시간 런 뒤에 999포인트가 모이지 않는 상황이 생길 수 있습니다.

필수 게임 설정:

| 메뉴 | 항목 | 값 |
| --- | --- | --- |
| 난이도 | 주행 보조 사전 설정 | 보조 최대 |
| HUD 및 게임플레이 | 기술 | 꺼짐 |
| HUD 및 게임플레이 | 다음에 할 일 | 꺼짐 |
| 비디오/그래픽 | 해상도 | FHD, 1920 x 1080 |
| 비디오/그래픽 | 프레임 제한 | 120 FPS 권장 |

위 설정은 권장이 아니라 정상 동작 전제입니다.
주행 보조가 다르면 이벤트랩 주행이 흔들릴 수 있고, `기술` 표시가 켜져 있으면 화면 위에 뜨는 알림이 인식과 입력 타이밍을 방해할 수 있습니다.
`다음에 할 일`이 켜져 있으면 오픈월드 복귀 후 이벤트 추천 카드가 떠서 구매 메뉴 진입을 막을 수 있습니다.
해상도가 FHD가 아니면 이미지 인식 기준 위치가 달라져 제조사/작업차 선택이 실패할 수 있습니다.
프레임도 이벤트랩 주행 속도에 영향을 줍니다.
FHD 기준 정상 주행 시간은 한 판 약 4분 20초 언저리입니다.
한 판이 5~6분대로 길어지면 튜닝이나 마스터리 문제로 보기 전에 프레임 제한을 120 FPS로 맞춘 뒤 다시 테스트하세요.

프로그램 쪽 준비:

- 첫 실행 기본값은 `스바루작`입니다.
- `마즈다작`은 카패스 DLC 차량을 쓰는 선택 프리셋입니다.
  본인이 해당 DLC 차량을 쓸 수 있는 경우에만 슈퍼휠 탭에서 직접 누르세요.
- `맵 코드`를 비우면 내 즐겨찾기 경로를 사용합니다.
- Xbox/Game Pass에서 가상 키보드가 뜨면 맵 코드를 비우는 방식을 권장합니다.
- 구매 횟수, 강화 횟수, 삭제 횟수를 길게 잡기 전에 작은 값으로 테스트합니다.
- 실행 중에는 게임패드, 키보드, 마우스로 다른 입력을 넣지 않습니다.
- 사용 로그와 인식 실패 캡처는 `subaruu.exe` 옆의 `log` 폴더에 저장됩니다.
- 가장 최근 실행 상태 요약은 `log/last-run-status.txt`에 저장됩니다.

문제가 생겨서 도움을 요청할 때는 `log` 폴더와 `subaruu.conf`를 같이 보내주세요.
로그에는 실행한 탭, 주요 설정값, 진행 상태, 중단 이유, 이벤트랩 인식 실패 캡처 경로가 남습니다.
`last-run-status.txt`에는 마지막 실행의 탭, 회차, 진행 단계, 마지막 정상 지점, 정지 사유가 요약됩니다.

## 5. 절대 시작하면 안 되는 화면

정상 구매 경로는 자동차 매장이 아닙니다.
정상 경로는 자동차 컬렉션입니다.

잘못된 화면:

- 자동차 매장 화면
- 상품권 구매 화면
- 스토어 연결 중 화면
- 현금 결제 관련 화면

이 화면에서 계속 진행하면 원하지 않는 구매 화면으로 들어갈 수 있습니다.
바로 `F2`를 누르고, 자동차 컬렉션으로 돌아가세요.
자동차 컬렉션 안에서 제조사를 고른 뒤 보이는 차량 선택 화면은 정상 경로입니다.

## 6. 탭별 시작 위치

### 스바루런

정상 시작 위치:

```text
이벤트랩 주행 화면
```

이미 이벤트랩 안에서 차량을 타고 주행을 시작할 수 있는 상태여야 합니다.
오픈월드에서 이벤트랩을 찾아가는 기능은 이 탭에 없습니다.
주행 중 가속 입력은 `W` 키만 유지합니다.
환경에 따라 동시 입력이 다르게 처리될 수 있어 기본 가속 입력을 단순화했습니다.

### 구매

정상 시작 위치:

```text
자동차 컬렉션 > 제조사 > 자동차 선택 화면
```

구매할 차량 카드에 커서가 올라가 있어야 합니다.
자동차 매장이나 상품권 화면이면 잘못된 경로입니다.

올인원/테스트에서는 오픈월드에서 자동차 컬렉션으로 들어가고, `제조사 이동횟수`와 `구매차 이동횟수`를 사용합니다.
단독 구매 탭은 현재 화면이 이미 맞다는 전제로 구매 입력만 반복합니다.

### 슈퍼휠

정상 시작 위치:

```text
페스티벌 > 경매장 > 경매 시작 > 차량 선택 목록
```

`경매 시작` 타일 위가 아닙니다.
차량 카드들이 보이는 목록 화면이어야 합니다.

슈퍼휠 탭은 최근 추가됨 기준으로 차량을 고르고, 탑승한 뒤 자동차 마스터리를 찍습니다.
테스트런/올인원에서 구매 후 `경매 시작` 대신 윗칸으로 들어가면 `경매장 대기`를 8~12초로 늘려 다시 확인하세요.
차량 선택 목록 검증에 실패하면 슈퍼휠과 삭제로 넘어가지 않고 멈추는 것이 정상입니다.

슈퍼휠 탭의 `스바루작`, `라부엘토작`, `마즈다작`은 프리셋 버튼입니다.
직접 실행 버튼이 아니라, 아래 값과 제조사/구매차 이동횟수를 한 번에 채우는 버튼입니다.

| 버튼 | 구매 | 강화 | 삭제상한 | 용도 |
| --- | ---: | ---: | ---: | --- |
| 스바루작 | 33 | 33 | 32 | 기존 SUBARU 작업 기준 |
| 라부엘토작 | 25 | 25 | 24 | 레부엘토 작업 기준 |
| 마즈다작 | 47 | 47 | 46 | 카패스 DLC 차량 보유자용 마즈다 작업 |

프리셋 버튼을 누른 뒤에도 바로 매크로가 실행되지는 않습니다.
값이 맞는지 확인하고, 정상 시작 위치인 `페스티벌 > 경매장 > 경매 시작 > 차량 선택 목록`에서 `F1`을 눌러야 실행됩니다.
배포 기본값은 `스바루작`입니다.
마즈다작은 포인트 효율이 좋지만 카패스 DLC 차량이 필요하므로, 일반 사용자 기본값으로 두지 않습니다.
프리셋 이동횟수는 현재 확인된 기준값이며, 매주 목요일 23:30 전후나 새 차량 추가 뒤에는 달라질 수 있습니다.

### 삭제

정상 시작 위치:

```text
페스티벌 > 경매장 > 경매 시작 > 차량 선택 목록
```

삭제 탭은 최근 추가됨 기준 1번 칸을 삭제 대상으로 삼습니다.
반드시 1번 칸이 삭제할 차량인지 확인하세요.
삭제는 되돌리기 어렵습니다.

### 테스트

정상 시작 위치:

```text
오픈월드
```

테스트 탭은 올인원과 같은 경로를 짧은 횟수로 검증합니다.
구매 위치, 작업차 선택, 슈퍼휠, 삭제가 본인 환경에서 이어지는지 확인하는 용도입니다.
테스트 탭의 `맵` 칸은 테스트런 전용 맵코드입니다.
비워두면 공유코드 입력창을 열지 않고 게임 안의 `내 즐겨찾기` 경로를 사용합니다.

### 올인원

정상 시작 위치:

```text
오픈월드
```

올인원은 이벤트랩 진입, 스바루런, 구매, 슈퍼휠, 삭제를 이어서 진행합니다.
가장 길고 위험한 탭이므로 `최소런`과 `권장런` 통과 후 사용하세요.

## 7. 구매 이동횟수 찾는 법

구매 위치는 사람마다 다를 수 있습니다.
DLC 보유 상태, 차고 상태, 신규 차량 추가 여부에 따라 같은 차량도 위치가 달라질 수 있습니다.

확인 순서:

1. 게임에서 자동차 컬렉션으로 갑니다.
2. 제조사 선택 화면을 엽니다.
3. 시작 커서 위치에서 목표 제조사까지 몇 번 눌러야 하는지 셉니다.
4. 그 값을 `제조사 이동횟수`에 넣습니다.
5. 목표 제조사에 들어갑니다.
6. 목표 차량까지 몇 번 눌러야 하는지 셉니다.
7. 그 값을 `구매차 이동횟수`에 넣습니다.

예시:

```text
제조사 화면에서 SUBARU까지 위 2번
SUBARU 차량 목록에서 목표 차량까지 오른쪽 3번
```

그러면 프로그램의 제조사/구매차 이동횟수에 같은 값을 넣습니다.

중요한 점:

- 공유된 값은 참고용입니다.
- 다음 주에 신규 차량이 들어오면 위치가 바뀔 수 있습니다.
- 구매 차량이 다르면 이동횟수부터 다시 확인하세요.

## 8. 슈퍼휠/삭제가 최근 추가됨 기준인 이유

차량을 여러 대 구매하면 방금 산 차량들이 최근 추가됨 쪽에 모입니다.
프로그램은 이 정렬을 기준으로 새로 산 차량들을 찾아갑니다.

그래서 작업 중에는 아래 행동을 피해야 합니다.

- 슈퍼휠스핀을 직접 까기
- 다른 신규 차량 받기
- 다른 차량 구매하기
- 차고 정렬을 수동으로 흔들기

중간에 새 차량이 섞이면 최근 추가됨 1번 칸이 예상과 달라질 수 있습니다.
그러면 강화 대상이나 삭제 대상이 꼬입니다.

## 9. Xbox/Game Pass 사용자

Xbox/Game Pass 환경에서는 공유코드 입력창에서 가상 키보드가 뜨는 경우가 있습니다.
이때 키 입력 포커스가 가상 키보드로 넘어가면 코드 입력이 꼬일 수 있습니다.

권장 방식:

1. 스바루작 이벤트랩 맵 `113938786`을 게임 안에서 즐겨찾기합니다.
2. 프로그램의 `맵 코드` 칸을 비웁니다.
3. 프로그램을 다시 열어도 맵 코드가 비어 있는지 확인합니다.
4. 테스트런으로 내 즐겨찾기 경로가 들어가는지 확인합니다.

맵 코드가 비어 있을 때 프로그램은 아래 흐름을 사용합니다.

```text
이벤트랩 검색/찾기 화면
PageDown 7회
내 즐겨찾기 선택
Enter
3초 대기
Enter
3초 대기
차량 선택 화면
```

즐겨찾기에 스바루작 이벤트랩 맵이 없으면 이 경로는 실패합니다.
여러 이벤트랩을 즐겨찾기해 둔 상태라면 순서가 꼬일 수 있으니, 스바루작 맵만 따로 즐겨찾기하는 것을 권장합니다.

## 10. 작업차 이미지 인식

작업차 선택은 이미지 자동 모드를 사용합니다.
이벤트랩 차량 선택값 `Pg`, `←`, `→`, `↑`, `↓`가 모두 0이면 이미지 인식으로 동작합니다.

현재 기준:

- 제조사 화면에서 SUBARU를 찾습니다.
- SUBARU가 이미 선택되어 있고 점수가 충분하면 이동하지 않습니다.
- 작업차 화면에서 기본 파란 1998 SUBARU 카드 이미지를 우선 찾습니다.
- 찾은 카드 위치까지 키보드 `↑`/`↓`로 이동합니다.
- 선택된 카드가 작업차인지 다시 검증한 뒤 `Enter`를 누릅니다.
- 키보드 선택 검증이 실패할 때만 마우스 클릭을 백업으로 사용합니다.

권장:

- 즐겨찾기에는 작업차 1대만 남기세요.
- 기본 파란 1998 SUBARU 상태가 가장 안정적입니다.
- 비슷한 SUBARU 차량을 여러 대 즐겨찾기하면 인식 변별성이 떨어질 수 있습니다.

## 11. 자주 터지는 문제와 복구법

### 자동차 매장/상품권/스토어 화면이 뜸

잘못된 경로입니다.
즉시 `F2`로 멈추세요.
정상 구매 시작 위치는 자동차 컬렉션의 차량 선택 화면입니다.

### 가상 키보드가 떠서 맵 코드 입력이 안 됨

맵 코드를 비우고 이벤트랩 즐겨찾기 방식을 사용하세요.
Xbox/Game Pass 환경에서는 이 방식이 더 안전합니다.

### 제조사 검색에서 멈춘 것처럼 보임

게임 로딩이 느리거나 화면 전환 전에 입력이 들어간 경우일 수 있습니다.
작업차 선택 후 이벤트맵 로딩에서 멈추거나, 스바루런 후 오픈월드로 나왔는데 구매 메뉴로 못 들어가면 `스바루런` 탭의 `로딩 대기`를 30~45초로 늘려보세요.
이 값은 이벤트랩 로딩, 스바루런 후 구매 위치 진입 전 대기, 보정런 후 오픈월드 복귀 대기에 쓰입니다.
기본값은 25초입니다.
빠른 PC에서는 테스트런으로 확인한 뒤 20초까지 줄일 수 있지만, 한 번이라도 월드 복귀 후 화면만 움직이면 다시 30초 이상으로 올리세요.
구매가 되지 않았는데 슈퍼휠/삭제 단계로 넘어가는 것처럼 보이면 즉시 `F2`로 멈추세요.

### 스바루런 후 이벤트 카드 화면이 뜸

게임 설정에서 `HUD 및 게임플레이 > 다음에 할 일`이 `꺼짐`인지 먼저 확인하세요.
이 설정이 켜져 있으면 오픈월드 복귀 후 이벤트 추천 카드가 떠서 구매 메뉴 진입을 막을 수 있습니다.
오픈월드까지 나온 뒤 구매 메뉴로 들어가는 첫 메뉴 이동 입력이 씹힌 경우일 수 있습니다.
이 경우 뒤쪽 `아래 방향키`와 `Enter` 입력이 오픈월드 메뉴에서 실행되어 이벤트 카드 화면으로 들어갈 수 있습니다.
최신 빌드는 구매 위치 진입 때 `Esc` 후 5초를 기다리고, 첫 왼쪽 이동 4회를 천천히 보냅니다.
반복되면 즉시 `F2`로 멈추고 `log` 폴더에서 `purchase-menu-open-wait` 기록을 확인하세요.

### 경매 시작 대신 윗칸으로 들어감

구매 후 경매장 화면이 뜨기 전에 입력이 들어가면 `경매 시작`으로 내려가는 방향키가 먹지 않을 수 있습니다.
`슈퍼휠` 탭의 `경매장 대기` 기본값은 8초입니다.
그래도 반복되면 12초로 올려 테스트하세요.
차량 선택 목록이 확인되지 않으면 프로그램이 슈퍼휠/삭제로 넘어가지 않고 멈춥니다.
이 경우 `log` 폴더의 세션 로그와 실패 캡처를 함께 확인하면 어느 화면에서 멈췄는지 볼 수 있습니다.
구매 후 경매장으로 들어가야 하는데 자동차 매장으로 빠지는 문제도 `경매장 대기`를 늘려야 합니다.

### 스킬포인트가 부족함

가장 먼저 작업차를 제대로 탔는지 확인하세요.
작업차가 아니면 포인트 수급이 크게 줄어듭니다.
프레임 제한도 같이 확인하세요.
FHD 기준 스바루런 정상 시간은 한 판 약 4분 20초 언저리입니다.
한 판이 5~6분대로 길어지면 프레임 제한을 120 FPS로 맞춘 뒤 다시 테스트하세요.
스바루작 33대는 990포인트가 필요합니다.
마즈다작 47대는 987포인트가 필요하지만, 카패스 DLC 차량을 사용할 수 있는 환경에서만 권장합니다.

### 구매 차량이 다름

제조사/구매차 이동횟수가 본인 화면과 맞지 않는 상태입니다.
본인 화면에서 제조사와 차량 위치를 다시 세세요.

### 삭제 대상이 꼬임

구매/강화 중간에 다른 새 차량이 섞였을 가능성이 큽니다.
작업 중 슈퍼휠스핀을 직접 까거나 다른 신규 차량을 받지 마세요.

## 12. 드라이런

게임 입력 없이 캡처 이미지만으로 인식 결과를 확인할 수 있습니다.

제조사 화면:

```powershell
.\subaruu.exe --eventlab-manufacturer-dryrun "C:\path\to\manufacturer-screen.png"
```

작업차 화면:

```powershell
.\subaruu.exe --eventlab-workcar-dryrun "C:\path\to\subaru-cars-screen.png"
```

결과는 콘솔과 `log/eventlab-*-dryrun-*.txt`에 저장됩니다.
인식 오버레이 이미지는 `log/eventlab-*-dryrun-*.png`에 저장됩니다.
